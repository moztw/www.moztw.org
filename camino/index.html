Camino: Mozilla 的威力、Mac 的風格

能代替 Mozilla Firefox 的另一個 Mac OS X 瀏覽器再進化！ Camino 1.0 增加了許多方便新功能、更快的效能、更佳的穩定性及改良後的佈局引擎。

使用 Camino 的理由

“It's brisk, baby.” - Om Malik of Business 2.0

“Best. Browser. Ever.”- Justin Collins

“Camino offers top-notch performance, elegant tabbed browsing, and an easy way to view and hide histories and bookmarks.” - John Rizzo, c|Net

世界最棒的佈局引擎
Camino 使用與 Firefox 相同的 Gecko 佈局引擎，不只顯示快速，還支援像 XML、XHTML、DOM、CSS 及 SVG 等最新 W3C 標準，和 QuickTime、Java、Flash、Shockwave 等 Plugins 也有更好的整合。
完美整合
Camino 和 Mac OS X 緊密結合。原生的 Cocoa 介面讓程式更有質感，符合整體操作環境。Camino 也能和 Spotlight、通訊錄、鑰匙圈、Finder、Dock、Bonjour、服務及系統偏好設定等一起協同使用。
廣告視窗控制
Camino 能終結擾人的彈出型視窗和廣告，讓您上網興致更佳。內建的廣告阻擋技術讓您在茫茫網海中不易分心。Camino 會在狀態列上顯示圖示，您有興趣的廣告也能立即顯示。
搜尋工具
現代網路瀏覽中，搜尋是最主要的功能。不論是整個網路或是單獨的網頁，Camino 都提供了有用的搜尋工具。工具列上能直接以 Google 或其他搜尋引擎搜尋，隨打即找讓您快速搜尋頁面中的文字，書籤管理員中也能搜尋瀏覽過的網站，在 Mac OS X 10.4 中更能用 Spotlight 搜尋。
隱私權及安全工具
Camino 讓管理或刪除您在網路上留下的資訊變得更簡單。新的「重置 Camino」 鈕能瞬間消去所有資訊。Camino 的隱私權設定能讓您簡單的管理儲存在電腦中的 Cookie。支援額外的認證管理，更允許您對安全連線做進一步的控制。
分頁瀏覽
分頁瀏覽讓您瀏覽網路變得更簡單。您可在單一視窗中開啟多個分頁，每個視窗中所開的分頁數也不受限制，甚至還能選擇要在前景或背景開啟網頁。
下載檔案
Camino 的檔案下載非常友善，可顯示所有下載檔案的狀態、速度及剩餘時間。還提供像暫停、續傳等進一步的下載控制。
開放原始碼
Camino Project 是 100% 的開放原始碼軟體。任何人都能瀏覽程式原始碼，有問題也能快速修正，這就是為甚麼 Camino 這麼穩定和安全的原因。

系統需求

  • Mac OS X 10.2 或更新版本
  • 128 MB 記憶體
  • 50 MB 可用的磁碟空間
  • Camino 1.0 已是 Universal 應用程式，在採用 Intel 與 PowerPC 處理器的 Mac 電腦上皆可執行。
