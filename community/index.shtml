Mozilla 全球社群

Mozilla 是一個全球社群，由世界各地的志工與員工：工程師、設計師、翻譯者、推廣者與使用者共同組成。我們依循 Mozilla 宣言，致力於打造更開放自由的網路、提升人們的網路體驗、宣揚開放源碼共享精神。我們推動開放標準，相信惟此才能確保任何人在何時何地都能享受最安全、流暢、多元的網際網路。

Many Voice, One Mozilla.Mozilla 由世界各地眾多社群、貢獻者及與各種專案計畫所構成，在此為您介紹正體中文的諸多社群與重點專案，歡迎自由探索與加入 Mozilla 這個廣大豐富的家族。

Mozilla 社群與組織

MozTW, Mozilla 台灣社群

Mozilla Taiwan Community

MozTW 是台灣的 Mozilla 志工社群，自 2004 年起即負責維護 Mozilla 軟體正體中文版本與 語言套件，提供 moztw.org 網站軟體下載與討論區。MozTW 本著推廣主力 Firefox 易學常用的特性，自許為使用者接觸開放、自由文化的起點，持續辦理自由文化相關課程與活動，與全球社群一同打造更美好的網路。

→ MozTW 首頁、 Telegram Group、 討論區、 Google Group、 連繫資訊

Mozilla Taiwan 美商謀智台灣分公司

Mozilla Taiwan

Mozilla 台灣分公司是 Mozilla 全球第九個辦公室，初期預計招募 40 位軟體研發人員，除肩負 Firefox OS 研發中心的重責大任，也會密切與在地的開放社群合作，建構行動上網的開放平台。

→ Mozilla Taiwan、 工作機會、 聯絡資訊

Mozilla Links 正體中文版 編輯機動隊

Mozilla Links 是清新歡樂的 Mozilla 新聞網，由翻譯機動隊全體志願者共同營運，每週精選全球社群、各 Mozilla 相關 Blog 及協力作者專文，將 Mozilla 專案的第一手資訊，編譯後再傳播給所有中文摩茲客及火狐迷。

歡迎加入翻譯機動隊行列，只要打開待譯資料夾即可立即開始翻譯文章。

→ 加入翻譯機動隊╱投稿

美商謀智 Firefox 校園大使

Mozilla Taiwan Firefox Campus Ambassador

Mozilla Taiwan 號召台灣青年學子加入 Firefox 校園大使的行列。透過成為 Firefox 校園大使，在校園內協助推廣 Firefox，成立 Firefox 社團，作為 Firefox 小幫手解決疑難雜症。

→ Facebook 社團

MozTW 字幕組

MozTW 字幕組致力於翻譯、製作 Mozilla 相關影片的正體中文字幕，將 Mozilla 的理念與影像紀錄傳達給更多中文用戶。

→ 待譯影片清單、 MozTW Youtube 頻道

Firefox 香港使用者社群

A community group of Firefox users living in Hong Kong. Join us now if you enjoys firefox for your web surfing, and and spread it to your friends.

Mozilla Reps Mozilla 代表計畫

Mozilla Reps 計畫的目標是致能與支持志工成為 Mozilla 地域性╱各語系官方代表，並提供簡易的框架與工具協助 Mozillians 組織活動、招募貢獻者。Mozilla Reps 透過跨國通力合作、支持在地社群，並促使 Mozilla 全球貢獻者成長與茁壯。

→ 台灣的 Mozilla Reps

Mozillians Mozilla 社群通訊錄

你身為 Mozillian，想要找些夥伴在高雄辦聚會？需要一個專業的 XPCOM 駭客幫忙審核某個 patch？想要獲得 Mozilla 基金會第一手消息？Mozillians 社群通訊錄就是個可以讓你尋找志同道合夥伴的地方，歡迎加入！

Mozilla 各專案

在此介紹 Mozilla 社群目前推動的各種專案，歡迎協助宣傳、分享與自由貢獻。

SUMO - Mozilla 技術支援 Mozilla Support

SUMO 是 SUpport MOzilla 的縮寫字，讓使用者透過搜尋功能，查詢 Mozilla 相關產品的說明文件。SUMO 中的說明文件主要由全球社群志工撰寫與翻譯，當使用者點選 Firefox 的「說明」→「Firefox 說明」時，就會開啟 SUMO 網站。

→ 更多介紹、 徵求超人義工

狐耳摩莎 Foxmosa Foxmosa

Foxmosa 是 MozTW 喜歡陪你在網路世界玩耍的吉祥物小狐狸，歡迎參觀他的個人網站喔。

Mozilla Links 正體中文版 Mozilla Links-zh

Mozilla Links 每週精選全球社群、各 Mozilla 相關 Blog 及協力作者專文，將 Mozilla 專案的第一手資訊，編譯後再傳播給所有中文摩茲客。

GFX.tw 抓火狐 Get Firefox

「抓火狐」是屬於您的 Firefox 推廣平台。既然你都可以把 Firefox 用擴充套件自訂成你喜歡的樣子，那為何不把你最喜歡的功能／套件，製作成專屬推廣網頁，宣傳給別人呢？立即到抓火狐來取得個人下載網頁。

MozTW 討論區 使用者技術支援 MozTW Forum

MozTW 討論區上天天都有使用者詢問問題、尋求協助，Mozilla 仰賴使用者間互助，作為產品支援最重要的一環。請幫忙回答使用者詢問的軟體使用問題，您不需向任何人報名，只要回答問題，就立刻成為 Mozilla 社群的一員了！

MDN - Mozilla 開發者社群 Mozilla Developer Network

Mozilla 開發者社群的目標是成為網路上最好、最詳盡的開發文件平台。包含 Mozilla 與 Open Web 所有相關技術的文件，以 Wiki 的方式供社群志工編撰、翻譯，不論哪個平台與瀏覽器的使用者，都可以加入貢獻。除開發文件外、還有許許多多精彩的 DEMO 作品及每月一主題的 Dev Derby DEMO 比賽。

Firefox 夥伴計畫 Firefox Affiliates

幫助我們推廣 Firefox 與其他 Mozilla 專案，打造一個更好的網路環境。歡迎透過 Firefox 夥伴計畫，在您的 Blog 或 Facebook 貼上 Firefox 貼紙，向世界分享你對 Firefox 的愛。

→ Firefox 夥伴計畫、 Firefox 夥伴計畫 - Facebook App

L10n - Mozilla 產品在地化 Localization

由於全球社群的協力，Firefox 與 Mozilla 網站共提供 80 種以上的語言版本，一同加入我們的在地化計畫，協助更多的使用者能用最熟悉的語言上網吧！
