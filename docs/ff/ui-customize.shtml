自訂介面

或許您一輩子也不會按到那顆「首頁」鍵，也或許您希望能將書籤項目換個地方擺，這些要求在 Firefox 裡都能輕鬆做到。

在工具列上按下滑鼠右鍵，選擇「自訂」，您便能將各種按鈕「拖」到工具列上；如果想移除某個按鈕，也只要從工具列上拖回自訂按鈕面版即可。

若是工具列不符您的需求，您也可以自行新增工具列。你的軟體介面，就該由你自己決定。

實際瞧瞧

想要盡可能的節省空間嗎？您可以將書籤項目「拖」到選單右方，再隱藏書籤工具列，就少佔一排空間囉！


依照您的習慣，輕鬆自訂工具列。
