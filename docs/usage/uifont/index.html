Updating Mozilla Font Size

問題: Mozilla 的字 (尤其是 MacOS/UNIX 版) 太小了! 所以很難看; 怎麼變大?
如果你覺得 Mozilla 字型太小，請下載 userChrome.css 並安裝後 修改裡面的設定。 基本上就是要用 userChrome.css 去一個個改掉你覺得太小的元件(如果改好歡迎寄給我 :D)。

不過如果你沒時間也不懂, 有個簡單的方法: 下載並安裝 Download 下載頁的 userChrome.css, 然後編輯它。在檔案的最後面有些我寫的註解，可以用來 快速指定字型大小 (不過這會改掉全部的元件!!)。請照上面的指示， 修改後重開 Mozilla 即可。
