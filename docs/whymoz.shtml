* Mozilla supports standard HTML, DOM and CSS. IE sucks.
ref: w3c pages, especially CSS
* Mozilla 對 HTML 及 CSS, DOM 的支援遠較 IE 完整。
請參考: w3c 網頁, 特別是 CSS 。 CSS的網頁本來是一個半透明的 menu 固定在 browser 右半邊不會被被捲動走， IE 看起來是不透明的醜 menu 在文件開頭左邊而且會捲走。 再來，該網頁用 Mozilla 可由「檢視」「使用 Stylesheet 」選用不同 style sheet，IE也不支援。還有，w3c 的網頁大多有 website navigation bar (檢視→顯示/隱藏→網站瀏覽列)，IE也不支援
Mozilla 1.0 的 Demo page 中的好網頁: http://www.meyerweb.com/eric/css/edge/complexspiral/demo.html , IE 又再次遜掉
