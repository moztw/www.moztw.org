從 Internet Explorer 改用 Mozilla Firefox

下載 Mozilla Firefox

  1. 您可至首頁點選與您作業系統相符的連結，以便下載 Firefox。
  2. 當出現詢問要「開啟」或「儲存」的對話方塊時，直接選擇「開啟」便可開始安裝。如果您欲將檔案存至硬碟中，可存放在桌面上，隨後雙點 Firefox Setup 1.0PR.exe 以便開始安裝。

您亦可自 Mozilla Store 購買 Firefox CD 及使用手冊。

輕鬆改用 Firefox

Firefox 可匯入您在 Internet Explorer 中的既存設定。在您第一次安裝 Firefox 時，匯入精靈將自動執行（也可以稍後從「檔案」 > 「匯入」啟動），並試著匯入 IE 的「我的最愛」、「Cookies」、「已儲存密碼」和其他種類的資料，您可因此節省依需求重新設定 Firefox 的時間。

瀏覽器 Plug-in、附加功能及工具列

許多您曾安裝於 Internet Explorer 的 plugin、附加功能及工具列也有為 Mozilla Firefox 設計的版本，包括：

  • Macromedia Flash Player、Apple QuickTime、 Real Networks RealPlayer、 Windows Media Player、Adobe Acrobat Reader以及 Sun Microsystems Java Plug-in
  • Google 工具列

此外，Mozilla Firefox 也可安裝五十種以上的 擴充套件 ，大大擴展 Firefox 的功能。

您會發現 Microsoft Internet Explorer 和 Firefox 有許多快速鍵功能相似。快速鍵的完整清單請參考 List of Firefox Keyboard Shortcuts。
