Subject: 有關安裝Activation啟動解決方案！
有很多人對Netscape 6.2安裝後，自動啟動Activation，因不明白其用意而非常感冒！ 事實上這是Netscape強迫使用者，在安裝軟體後能馬上申請免費成為其 「會員」。若你成為Netscape會員後，就不會再有自動啟動Activation的麻煩。
其實這個動作大家可以想像申請BBS帳號一樣，先註冊你的姓名、綽號、密碼，然後BBS伺服器會發出一封e-mail確認信要求你回信，當你回信後，BBS伺服器才真正完成你的註冊及使用權！
因此當你安裝完Netscape 6.2後，Netscape伺服器會問你是不是Netscape會員，若不是就要申請；當填寫完後Netscape會問你喜歡何種ScreenName，如我輸入的綽號為Love，它可能給你ScreenName為LoveN，看你要不要；一旦輸入完成，Netscape伺服器會發出一封e-mail確認信要求你回信，而你回信時只要在第一行鍵入「OK」然後回信即可完成Netscape會員動作！下次重新安裝只要輸入會員帳號密碼即可。若你的密碼忘記了！可至Netscape網站查尋密碼！
其網站為http://ureg.netscape.com/iiop/UReg2/login/login
