Thunderbird 是一套快速、安全而簡單的全功能電子郵件、RSS 和新聞群組管理程式。其它關於 Thunderbird 的訊息，請參考 Thunderbird 產品首頁。

此版本資訊包含了 Thunderbird 1.5 的新鮮事、下載與安裝指示、已知問題以及其他連結等訊息。如發現任何程式錯誤，請先行將版本資訊與錯誤回報注意事項閱讀完畢，再到 Bugzilla 回報。

我們想知道您對 Thunderbird 的意見。歡迎到討論區討論相關議題：您可至 MozillaZine 上的英文討論區或 MozillaTaiwan 的Thunderbird 中文討論區與我們分享意見。

Thunderbird 1.5 新鮮事

Thunderbird 1.5 是一套獲得無數獎項的電子郵件程式。

以下是值得一提的 Thunderbird 1.5 新鮮事：

  • 全新的線上更新功能會用更明顯的方式提醒用戶有新的更新，且所需下載的檔案大小可能不到 0.5 MB。此外擴充套件的更新方式也更加友善。
  • 郵件地址自動完成功能現在會依您寄信給對方的次數排序。
  • 即時拼字檢查。
  • 已存搜尋夾現在能跨多個帳號搜尋。
  • 內建防詐騙偵測功能，保護用戶不受詐騙郵件的侵擾。
  • Podcast 和其他 RSS 功能增強。
  • 從郵件中刪除附件。
  • 與伺服器端的垃圾信過濾機制整合。
  • 從郵件過濾器中直接回覆或轉寄郵件。
  • Kerbero 認證。
  • 撰寫郵件時自動儲存草稿。
  • 自動刪除郵件。
  • 全域資料夾的過濾器。
  • 產品使用體驗的改進，包括重新設計的選項介面和 SMTP 伺服器管理等。
  • 其他的安全性修正。

下載與安裝

系統需求

在安裝 Thunderbird 前請先確定您的電腦合乎系統需求。 從 1.5 開始，Mac OS X 用戶必須在 MacOX X 10.2 以上版本運行本程式。

下載 Thunderbird

Mozilla 基金會提供 Windows 版、Linux 版和 Mac OS X 版的 Thunderbird。

Windows: 安裝程式

Linux: GTK2 + XFT （無安裝程式）

Mac OS X: 壓縮磁碟映像檔

其他 Mozilla.org 未提供的系統或語系，請閱讀文件最後的「非官方建製版」一節。

安裝 Thunderbird

下載 Thundbird 安裝程式或壓縮檔後，請依下列步驟進行安裝：


Windows

以滑鼠左鍵雙點 Thunderbird Setup 1.5.exe 安裝程式圖示以開始安裝。

Mac OS X

下載 Thunderbird 1.5.dmg 檔後，請雙點 Thunderbird 磁碟映像檔圖示以便開啟於 Finder 中，隨後將 Thunderbird 應用程式拖曳至您的硬碟中。請不要直接雙點磁碟映像檔中的圖示！一定要將 Thunderbird相關檔案拖進硬碟後才執行。您亦可將圖示拖曳至 Dock 上方便取用。

其他系統

解壓縮下載回來的壓縮檔後執行 thunderbird

移除 Thunderbird

欲移除 Thunderbird，請依下列步驟操作：

Windows

選擇「開始＞控制台」，當控制台出現時，請以滑鼠左鍵雙點「新增或移除程式」， 然後在清單中找到「Mozilla Thunderbird (1.5)」並按下「移除」。

Mac OS X

將 Thunderbird 程式拖曳至垃圾桶。

其他系統

移除 thunderbird 資料夾。

上述步驟將保留您的個人設定檔，供日後重新安裝時使用。如果您想移除 Thunderbird 的個人設定檔資料夾，請依循「個人設定資料夾」一節的步驟。 請注意：這麼一來您將失去所有信件、通訊錄、設定及其他資訊，重新安裝 Thunderbird 時就必須重新建立個人設定。

其他資訊

個人設定資料夾

Thunderbird 將您的個人設定檔存放在下列位置之一：

Windows 2000, XP Documents and Settings\<UserName>\Application Data\Mozilla Thunderbird
Windows NT WINNT\Profiles\<UserName>\Application Data\Mozilla Thunderbird
Windows 98, ME Windows\Application Data\Mozilla Thunderbird
Mac OS X ~/Library/Thunderbird
Linux 及 Unix 系統 ~/.thunderbird

已知問題

以下列出 Thunderbird 1.5 部分已知問題，請閱讀後再回報新的程式錯誤，並請定期檢查本欄的更新內容 （如在此版本發現新問題，我們會隨時在英文網頁更新此欄）。

擴充套件與佈景主題

您可能會在我們新設計的擴充套件相容和更新機制上遇到問題。下面列出了已知三個問題的簡單解決方法。

  • 當您執行 Thunderbird 安裝程式後，沒有出現擴充套件相容性和更新精靈的話，您可以打開「擴充套件管理員」，手動檢查更新檔。
  • 如果擴充套件相容性和更新精靈有出現，但顯示不完全，請先試著點選其他視窗後再點選回該視窗。如果還是沒有解決問題，按一下「取消」後，打開「擴充套件管理員」手動檢查更新檔。
  • 如果擴充套件相容性和更新精靈下載並安裝了更新檔，但是安裝失敗的話，請打開「擴充套件管理員」再次檢查有無更新檔。

在 Thunderbird 1.0 安裝的擴充套件，到了 Thunderbird 1.5 時可能並不相容。這時您需要檢查該套件有無更新檔，使其可在 Thunderbird 1.5 上運作。如果您喜歡的擴充套件或佈景主題未能相容新版，請寫信給作者，鼓勵作者更新。當您升級到 Thunderbird 1.5 時，所有的擴充套件及佈景主題會自動停用，直到 Thunderbird 發現 1) 此套件原本就相容於 Thunderbird 1.5 或 2) 有新版套件可相容於 Thunderbird 1.5。這是因為 Thunderbird 每一版都有所更新，與擴充套件的整合模式也不盡相同，如果使用舊版套件反而可能發生問題。

所有系統

  • 安裝 Thunderbird 1.5 前，請確保您安裝在新目錄中，切勿直接覆蓋任何舊版 Thunderbird。
  • 如果您將 Thunderbird 安裝於權限有所管制的區域，那麼就得先由擁有權限的使用者啟動一次 Thunderbird，以便建立應有的初始啟動檔案。若由權限不足的使用者先行啟動，則可能會因為無法完整建立初始啟動檔案而不斷重新啟動。
  • 預設的拼字檢查字典只有英文。想加入其他非英文的字典，請閱讀 the 安裝說明。
  • 在全域收件匣和 POP3 帳號的收件匣之間切換時，請確定已將郵件備份到其他資料夾。
  • 若您未擁有 Thunderbird 安裝處的檔案寫入權限，軟體升級程序便無法進行，這是因為軟體升級必須於安裝處替換或建立檔案。

Windows

  • 以權力受限的一般使用者身份安裝 Thunderbird 時，就算安裝在可寫入的地方，有時依然會發生些許問題（像是預設郵件程式設定錯誤等），但程式還是可以用。當以受限使用者身份安裝時，請不要在未移除舊版的情況下指定安裝於同一資料夾，否則可能無效。
  • Windows 用戶只要安裝 Thunderbird 版的 Palm 同步擴充套件 就可以與 Palm 裝置互相同步通訊錄。 將檔案下載回來之後儲存，然後從 Thunderbird 的「擴充套件」視窗中安裝這個擴充套件。此套件安裝後將不會在擴充套件列表中顯示，安裝之後記得重新啟動 Thunderbird 套件才會有作用。

Mac OS X

  • 不要由磁碟映像檔中直接啟動 Thunderbird！這麼一來 Thunderbird 可能會不斷重新啟動（症狀是 Thunderbird 圖示在 Dock 中不斷跳起、消失再出現）。若要從不斷啟動的情形中終止 Thunderbird，請開啟 終端機（Terminal）後輸入「killall thunderbird-bin」並按下 Enter。之後請將 Thunderbird 安裝於有權寫入的位置、再試一次。若安裝於多人共用的電腦上，可以管理員身份安裝於共用位置，執行一次後所有使用者就都能用了。

Linux 和 Unix 系統

  • 在 Fedora Core 3(FC3) 中，GNOME integration 並不能適當的運作。 FC3 的使用者需要安裝 linc-1.0.3-3.1.i386.rpm。安裝好這個 RPM 後，在您安裝 Thunderbird 的目錄下執行以下的命令(您需要有寫入的權限)：

    touch .autoreg

    下一次您啟動 Thunderbird 時，GNOME integration 將會正常的運作。

 

常見問題集

  1. 我該怎麼幫忙呢？

    我們需要大量曝光。讓推廣 Thunderbird 成為您的任務，使您的親朋好友上司下屬都認識它： 如果您是個學生，就讓它廣佈校園之中；寫篇文章投稿至Slashdot等新聞站台上，發布 Thunderbird 新版的新聞；也可以在您的 Blog 上透漏一點相關訊息； 總之，儘可能的在網際網路中散佈，就像 Kazaa 的威力一樣，處處推廣 Thunderbird！

  2. 我使用其他瀏覽器，因為…

    如果您不願使用 Thunderbird，請告訴開發團隊您的理由。我們會持續注意Thunderbird forums中的使用者意見。

  3. 你們為什麼都沒有回信？

    請使用討論區， Thunderbird 團隊會定期閱讀上面的文章。我們常常都收到大量的信件，而您的可能不小心漏失了。

  4. 我該去哪裡取得新的佈景主題與擴充功能？

    擴充套件及佈景主題可自Mozilla Update下載。

  5. 哪些人正致力於 Thunderbird 的開發工作？

    很多人。詳細列表請參考「說明＞關於 Thunderbird」
