Thunderbird 68.0 系統需求

Windows

作業系統版本
  • Windows 7
  • Windows 8
  • Windows 10
  • Windows Server 2008 R2
建議硬體需求
  • Pentium 4 或更新處理器，需支援 SSE2 指令集
  • 1GB RAM
  • 200 MB 硬碟空間

GNU/Linux

軟體需求

請留意：各 GNU/Linux 發行版隨附的 Thunderbird 套件可能有其不同需求。

  • Thunderbird 需要以下函式庫才能執行：
    • GTK+ - 3.4 或以上版本
    • GLib - 2.22 或以上版本
    • Pango - 1.14 或以上版本
    • X.Org - 1.0 或以上版本（建議使用 1.7 或以上版本）
    • libstdc++ - 4.6.1 或以上版本
  • 我們建議安裝以下函式庫或套件，以取得最佳效能：
    • NetworkManager - 0.7 或以上版本
    • DBus - 1.0 或以上版本
    • GNOME - 2.16 或以上版本

Mac

作業系統版本
  • Mac OS X 10.9
  • Mac OS X 10.10
  • Mac OS X 10.11
  • Mac OS X 10.12
  • Mac OS X 10.13
  • Mac OS X 10.14
