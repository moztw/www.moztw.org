Firefox 3 RC2 中文測試版

Firefox 3 RC2 是給開發者預覽測試的新版 Firefox 瀏覽器，僅供測試使用。如果曾安裝最新的正式版 Firefox，之前安裝的附加元件非常可能無法於此測試版正常使用。本測試版可讓您認識對 Firefox 3 的模樣及功能，但仍有新功能尚未加入，且許多部分仍會修改補強。

如發現任何問題，非常歡迎您到 MozTW 討論區 與我們參與討論或是直接到 Bugzilla 回報錯誤訊息。

Firefox 3 Beta 4 已知問題

目前未及翻譯 RC 2 已知問題，請前往版本資訊原文閱讀，或閱讀以下 Firefox 3 Beta 4 的問題做為參考。(如您願意協助翻譯，請寄信給柏強。)

Firefox 3 Beta 4 以新版 Gecko 1.9 網頁佈局引擎構成。研發此全新的引擎耗時 31 個月，改善了超過一萬兩千個大小問題。以下簡要列出此版本的已知問題，請閱讀後再回報新的程式錯誤。新功能及其他詳細內容，請參考版本資訊

所有系統
  • Windows Live Mail 無法使用; 必須切換到 Hotmail 舊版介面 (bug 396259)
  • 新版的 GMail 聯絡人無法顯示 (bug 415252)
  • 「回報偽造網頁」選項目前暫無法使用 (bug 420677)
  • Firefox 不會問使用者是否要更改既存密碼 (bug 421058)
  • MathML 功能在 Firefox 3 Beta 4 尚未完全穩定 (bug 413115 及 bug 363240)
  • 當使用 Window-Eyes 時，新的網址列、加入書籤對話框與下載管理員通知的訊息不一致，請使用 JAWS 8.0 或其他螢幕閱讀程式來測試 Firefox 3 Beta 4 (bug 393398)
  • 續傳檔案系統會因為伺服器不允許重新連線時發生錯誤 (bug 401846)
  • 以 text/html 型態傳來的資料來源無法正確載入 (bug 381357)
  • 某些附加元件安裝後不會顯示「重新啟動」鈕 (bug 414936)
  • 部份採用已棄用程式碼的附加元件無法正確安裝 (bug 406807)
  • 個人隱私 > 瀏覽紀錄 > 保留瀏覽紀錄裡的網址設定為 0 時沒有作用 (bug 366075)
  • DOM 觀查器已被移除，以額外的擴充套件形式提供。
Windows 及 Windows Vista
  • Windows Vista 及某些其他版本Windows並沒有提供一個Windows Media Player (WMP) 的plugin。必須透過一些步驟才能安裝plugin及開啟Windows Media檔案。安裝完畢後可能也需要做Windows Updates才能讓plugin正常地顯示這些檔案。
  • 若在 Windows 網際網路選項中設定為高安全性，執行檔下載後會被立即刪除 (bug 416683)
Mac OS X
  • 網站認證對話方塊的「更多資訊」鈕無法正確叫出網頁資訊視窗 (bug 415704)
Linux 及 Unix
  • Ubuntu 7.10 的使用者需要更新認證才能送出當機報告 (bug 407748)
  • 多功能滑鼠中，控制上、下頁的按鍵必須重新設定 (bug 420294)
