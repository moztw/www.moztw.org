Netscape Navigator 已停止更新

Netscape Navigator 自 2008 年起不再推出新版。我們推薦您改用 Firefox 或 SeaMonkey：

  • Firefox: 重新體驗網路

    Firefox 是多次獲獎的 Mozilla 新生代網頁瀏覽器。

  • SeaMonkey: 完整的 Internet 方案

    SeaMonkey 是全能的網路瀏覽工具。內含網頁瀏覽器、郵件軟體、IRC 聊天工具，以及網頁編輯功能。

您還可以為您的 Firefox 或 SeaMonkey 安裝 Netscape 佈景主題。

下列內容僅供參考。基於安全疑慮，我們不建議您使用舊版軟體做為日常使用。

Netscape 非官方正體中文版 (Netscape 7.2)

由於Netscape (網景) 系列目前沒有 Traditional Chinese 正體中文版的發行，所以臺灣 Mozilla 中文化小組 (moztw.org) 為了推廣 Mozilla 與 Netscape 特別製作了這邊的中文版。

中文版的製作是基於使用與 Mozilla 本地化同樣的技術， 並沒有重新編譯任何程式碼， 只有修改語系資源檔以及修改安裝程序檔 (script)。

但由於未取得 Netscape 公司正式的授權， 所以請記得在這裡取得的版本僅供測試， Netscape 本身以及實際重製的 moztw.org 並不為這些版本提供任何保障。

Netscape 8

Netscape 目前的最新的版本是8.1.2, 目前官方沒有發行任何中文版

Netscape 8 是以 Firefox 1.0.x 為基礎，有安全核心。也保證網頁不會版面或功能亂掉，即使這些網頁是專是為 IE 設計。使用開放原始碼和支持標準的 Firefox 1.0 核心，網站控制允許你調整安全級（Java、JavaScript、cookies 選項）允許你在必要時切換使用 IE 核心給某網頁。

Netscape 8 目前只有 Windows 版本。

有問題請利用 Netscape 討論區，不過如前所言， MozTW 還是以 Mozilla 相關產品為主。 Netscape 只算暫時的支援， 所以請保持您的耐心，謝謝。
