摩茲工寮 | Mozilla Community Space Taipei

摩茲工寮 - Mozilla Community Space Taipei 是為社群所設計、也由社群共同管理的空間。我們有桌椅、電力與網路、激發創意的小物、飲料，當然還有許多的 Mozillians，聚集在此討論、研究 Mozilla 與開放文化等領域，為自己熱愛的 Open Web 加柴添火。

地址：10460 台北市中山區德惠街23號地下一樓

Address: B1, No.23, Dehui St., Zhongshan Dist., Taipei City 10460, Taiwan

近期活動

　


開放時間

每週一至四晚間 19:30-22:00，不含每周五晚上的 MozTW Lab 台北場。詳細時間與特別活動請見左欄時間表。

如果你不確定有沒有人在，可在 Mailing List / IRC / Line / Telegram 上面問一下。現場有網路、電、飲料，歡迎大家來做 Mozilla / Open Source / Web 等等相關的事情 :)

我們的場地提供：

  • 雙向 100M 的無線網路
  • 很貼心的三孔插座
  • 讓你開心的碳酸飲料
  • 保證會涼的空調
  • 超棒的桌椅
  • 滿滿的熱情
