Firefox App/套件開發小趴

活動簡介

這是一個大家一起研究如何撰寫 Firefox OS 下 Open Web App 的 hackathon 。歡迎大家組隊參加！Firefox OS 是一個基於瀏覽器的手機作業系統，能讓開發者透過HTML、CSS、Javascript 去撰寫內部的應用程式。

感謝 慶隆商務俱樂部 贊助場地部份費用，如有需要租借活動場地、虛擬辦公室、私人招待所等服務可以上 clbc.tw 了解服務細節。
