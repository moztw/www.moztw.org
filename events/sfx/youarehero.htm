Firefox 英雄帖

  • 你是不是一上網就會逛很久，不知不覺工作列就被 IE 佔滿了？
  • 你是不是看簡體字網站看得眼睛痠？
  • 你是不是受不了關不完的廣告視窗？要不然就是看飛來飛去的 flash 廣告看到煩？
  • 你希不希望上網的經驗能更美好，更隨心所欲，更不受打擾？
如果以上皆是的話，那你一定要試一試連華爾街日報都推薦的 由 Mozilla 軟體基金會偕全球自由軟體社群共同打造的 Firefox 瀏覽器！

Firefox 1.0 正式版即將於 2004/11/9 現身江湖，你準備好了嗎？
你現在就可以先下載 Firefox 1.0 preview 版來嘗鮮：http://moztw.org/firefox/
  • 試試分頁瀏覽和不受廣告視窗干擾的感覺，再上 Firefox 討論區 看看別人的使用心得，你會覺得，世界真是太美好了!
  • 你如果覺得 Firefox 實在太棒了，請多多推薦給身邊的人使用！
  • 如果你能促使你的學校、公司、機構採用 Firefox 為預設瀏覽器的話，你就是我們的放火英雄！
