什麼是 Mozilla Firefox Party？
Firefox 1.0 正式版即將出爐，這對於 Firefox 而言是一個很重要的里程埤，Mozilla 官方為了慶祝這歷史性的一刻， 決定邀請全世界一同來辦一場屬於 Firefox 的 Party，於是展開了全球的串聯，這樣盛大的聚會，我們台灣怎麼可以缺席呢? 因此，台灣 Mozilla 社群決定在台灣辦一場盛大的聚會，並且迎接全台所有未加入 Firefox 行列的瀏覽器使用者，就讓我們一同來傳遞 Firefox 的聖火到全台灣吧!!
