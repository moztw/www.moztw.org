活動剪影

影片簡介

「如果你對軟體業感興趣，希望了解程式設計師的生活，以及 Mozilla 專案的起源和創造它的人們，
那麼千萬不要錯過這部紀錄片。我保證這是一部過目難忘的作品。」

——阮一峰

〈Code Rush〉是 David Winton 導演於 1998 年至 2000 年間拍攝的紀錄片，紀錄了矽谷的 Netscape 工程師們將瀏覽器原始碼釋出，成為 Mozilla 專案的經過；同時 Netscape 也正經歷被 AOL 併購的過程。本片描繪了 Netscape 的工程師們，犧牲日常生活與家庭，努力防止他們的公司倒閉的經歷。

2002 年釋出的 Mozilla 1.0 歷經十年演進，最終成為現今的 Firefox 瀏覽器，引領網路發展、帶來世界變革。明年是 Mozilla 基金會成立十週年及 Mozilla 計畫 15 週年。在這個新世代行動瀏覽器大戰的起點，就讓我們一起回顧 Mozilla 的起源，迎接未來的挑戰。

歡迎對自由軟體、開放原始碼感興趣，熱愛網際網路及 Firefox 瀏覽器的你，千萬不可錯過 MozTW 軟體自由日《Code Rush》放映會！
