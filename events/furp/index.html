活動簡介

從 4.0 開始進入快速更新週期的 Firefox 12，
與每半年一版，兩年一長期支援的 Ubuntu 12.04 LTS Precise Pangolin，

為了響應如此難得的機會，
MozTW 與 Ubuntu-TW 將在台中聯合舉辦 Party 啦！

活動資訊

  • 時間：5/27 下午 2:00（1:45 開始進場）
  • 地點：水舞饌中港店（台中市西區中港路一段121號）
  • 費用：200 元

內容

  • 吃吃喝喝是一定要的啊！
  • 各路熱血好手，分享各種有趣的主題。
  • 學習 ubuntu 及 Firefox 使用技巧
還可以
  • 抓人幫你解決 ubuntu 及 Firefox 的問題
  • 認識新朋友
  • A 紀念品
  • 族繁不及備載

交通

  • 在朝馬下車者可到中港路上搭乘 48、57、83、86、87、88、106、146 到民權路口站。
  • 在台中火車站下車者可到統聯客運搭乘 83、86、87，
    或是到車站正對面的台中客運總站搭乘 57、88、106 和 146 到民權路口站，
    也可以到南投客運轉運站搭乘 81 到中港路口站。
  • 只要有行經台中港路的公車，都可以在「中港民權路口」下車。
