Mozilla Suite 已停止更新

Mozilla Suite 自 2005 年起不再推出新版，下列軟體或許能滿足您的需求：

  • SeaMonkey: 完整的 Internet 方案

    SeaMonkey 繼承 Mozilla Suite，作為下一代的 Internet 套裝軟體。目前已有正體中文版，強烈建議您升級使用。

  • Firefox: 重新體驗網路

    Firefox 是多次獲獎的 Mozilla 新生代網頁瀏覽器。

  • Thunderbird: 改善你的信箱

    Thunderbird 是 Mozilla 新生代的 email 程式。 使用 Thunderbird ，讓你發送信件更快、更安全，也更簡單。

下列內容僅供參考。基於安全疑慮，我們不建議您使用舊版軟體做為日常使用。

網頁瀏覽

分頁瀏覽讓你對瀏覽網頁有全新的體驗。 你不用再每次開一個網頁就要開一個視窗弄得工作列滿滿的看不清楚， 也不用在每次點選只能開一個網頁。 分頁瀏覽配合書籤， 可以一次把一組網頁全部一起打開，連首頁也可以設定為好幾組網頁！

拒絕彈出型廣告視窗 讓你免受廣告之苦。 新的偵測程式在遇到網頁有彈出型視窗時會警告你， 你也可以對每個網站個別設定是否接受彈出型視窗。

圖片管理員讓你可以設定哪些網站不顯示圖片， 有效加速載入網頁，也可以減少廣告圖片。

隨打即找 方便你快速瀏覽頁面。 不用先叫出任何對話窗，直接打字， Mozilla 會在頁面中幫你找到 所打入字串的鏈結或文字。

除了上面提到的特點外，Mozilla 也具備了所有現代瀏覽器該有的功能： 密碼管理員、下載與 Cookie 管理員、佈景主題、 多國語言界面、多重平台支援、最新的網頁標準......

電子郵件與 Newsgroup

垃圾郵件控制 幫助你減少廣告商寄來的垃圾信。 Mozilla 的偵測引擎會利用你收到的信件自動學習，分辨效果會越來越好。

利用自訂的標籤(Labels) 與 郵件檢視方式 來管理你的電子郵件. 不同優先順序的郵件可用不同的顏色顯示。 排序後可方便你更快速的處理郵件。

Mozilla 支援 同時使用多組帳號 以幫助你在同一個界面下組織與分類郵件。

Mozilla 同時支援 S/MIME、收執回函、 LDAP、數位簽章等企業級安全標準。

設計師

Mozilla 的 HTML 編輯器 正在持續改進中，目前對動態圖片、表格縮放、快速插入與刪除儲存格、 加強的 CSS 、定位圖層 (layer) 等等都已有不錯的支援。 從簡易的文件到複雜的網頁專案，設計師都是你的好選擇。
