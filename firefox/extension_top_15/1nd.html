Firefox 擴充套件 Top 15

在你心目中最棒的 Firefox 擴充套件是哪些呢？現在就來看看社群成員在 2005 年一月共同推舉的十五個最佳幫手！

想親自試試嗎？勾選套件的「列入安裝清單」後，按下頁尾的「安裝這些套件」便可一次安裝喔！您必須以 Firefox 觀看本頁才能安裝這些套件。

配合 Firefox 1.5 釋出，此處套件更新為支援 1.5 的新版本。

NO.1 New Tong Wen Tang 新同文堂 0.2.7.9

  • 套件網站
  • 相關討論
  • 原生中文
「同文堂真是看不習慣簡體字者的救星」--社群成員 leolinfly

同文堂能把網頁上的簡體字轉為繁體（或繁體轉簡體）。此處提供由 softcup 與 passerby 更新改版的「新同文堂」，此版不再需要 jslib 即可運作。

NO.2 FlashGot 0.5.9.99

  • 套件網站
  • 配合 NetTransport
  • 配合 FlashGet
  • 配合其他軟體
  • 原生中文
「Firefox 終於可以方便地使用右鍵輕鬆下載檔案！FlashGot，我不能沒有你啊！」--社群成員 coolcd

這是 FlashGet 等下載軟體的遙控器，安裝後可自動判斷您的下載程式。此後欲下載檔案時，按下滑鼠右鍵就可以看到「使用XXXX下載此鏈結」等選項！亦有「下載選取部份包含的所有鏈結」及自行加入參考頁等功能。

適合安裝了 FlashGet 等下載軟體的人

NO.3 IEView 1.2.7

  • 套件網站
  • 相關討論
  • 原生中文
「IEView 使本人可以在大部分時間都用 Firefox 瀏覽，直到某些 IE Only 網站再『勉為其難』用 IE 開啟」--社群成員 zero_ngan

有某些網站還是只支援 IE，安裝這個套件可以讓你的 Firefox 右鍵選單多出一個選項，選擇後便可直接以 IE 開啟目前網頁。有鑑於國內網頁設計的標準風潮尚未吹起，這算是過渡時期不得已必備的選擇 ^^;

適合 Windows 作業系統

NO.4 Adblock Plus 0.5.11.2

  • 套件網站
  • 相關討論
  • 原生中文
「Adblock 是我個人認為 Firefox 最重要的套件之一」--社群成員 路人乙

Adblock 可以幫你檔掉你不想要看到的東西，例如圖片、Flash、Script、Frame 甚至 Popup。你可以自己設定要擋掉的網域跟內容。此處提供愛好者修改的 Plus 版本，新增白名單功能。

NO.5 All-in-One Gestures Extension 0.17.3

  • 套件網站
  • 相關討論 (1)
  • 相關討論 (2)
  • 原生中文
「有了它，幾乎所有動作都可以滑鼠手勢解決。」--社群成員 hemiola

按住滑鼠右鍵劃出上下左右，就能夠控制 Firefox 的許多功能(如上一頁、下一頁等)，也可自訂滑鼠手勢的功能。特別推薦想學哈利波特揮舞魔杖耍法術的朋友使用。

NO.6 Tab Mix Lite 1.5.10

  • 套件網站
  • 相關討論
  • 原生中文
「她讓 FireFox 的分頁瀏覽功能發揮得淋漓盡致！」--社群成員 zero_ngan

有了 Tab Mix，我們毋須再安裝許許多多分頁相關的套件，因為她已經整合了其他套件最受推崇的功能。Tab Mix 就是這麼一個小巧簡單 且功能全面的分頁瀏覽套件。此處提供的 Tab Mix Lite 為套件作者提供的新版本。

NO.7 Super Drag and Go 0.2.4

  • 套件網站
  • 相關討論
  • 另有中文版
列入安裝清單：
「推薦這個套件的拖曳圖片存檔功能，太方便了！」--社群成員 BobChao

就如同 Maxthon (myIE2) 的「超級拖放（Super Drag and Drop）」功能，用滑鼠拖拉後放開便可達到搜尋、存圖、開啟純文字網址、開新分頁等功能。

NO.8 Right Encoding 0.2.1

  • 套件網站
  • 相關討論
「進很多網站編碼不對，都要辛苦地去檢視裡找這個設定，現在點一下右鍵就得了！」--社群成員 Chris

當網頁編碼顯示錯誤時，原本只能從「檢視／語系及字元編碼」裡加以調整，加裝這個套件以後便會直接在右鍵選單中放入語系及字元編碼選單，方便點選。

NO.9 Tabbrowser Extensions 2.0.2005121301

  • 套件網站
  • 相關討論
  • 另有中文版
列入安裝清單：
「即使有這麼多缺點，說實在這個套件還是我安裝 Firefox 第一個會安裝的套件，功能真的太強大了。」--社群成員 ghost

這個分頁瀏覽相關套件有分頁群組、分頁自動重新載入、自訂中鍵等包羅萬象的功能，號稱是所有分頁套件中最強也最完備的套件。目前對於效能等問題爭議頗多，但依然是許多網友心中的第一名。

適合進階使用者

NO.10 ScrapBook 0.18.5

  • 套件網站
  • 相關討論
  • 原生中文
「逛網拍、看到好文時，狂抓 － 真幸福呀！￣▽￣」--社群成員 Chris

堪稱知識工作者必備套件，方便您擷取全部或部分網頁資料，並有方便的註解、標記、搜尋等功能。擷取下來的資料也可以離線閱讀或匯出供他人使用。

NO.11 Web Developer 0.9.4

  • 套件網站
  • 相關討論
  • 原生中文
「對網頁寫作者來說，這是一個相當好用的套件；就算不會寫，衝著可以關閉網頁裡的各種元件也值得一試！」--社群成員 raphael

這是一個為網頁設計師量身訂做的套件，可以隨時關閉圖片、JavaScript、CSS 樣式等效果，也可以輕鬆檢閱別人寫的 JavaScript 與 CSS，還有更改網頁尺寸、標示區塊元素等功能。

適合網頁設計師使用

NO.11 ForecastFox 0.8.2.5

  • 套件網站
  • 相關討論
  • 原生中文
「從此不再漏失天氣的資訊，對於平常生活或工作外出都是一個很好的功能」--社群成員 bise

假如你想知道目前的天氣，可以到氣象網站查詢。不過為了查個天氣特地連到氣象網站也算是大費周章，只要安裝這個套件，Firefox 的狀態列上就可以顯示最近幾天的天氣預報喔！

NO.13 Sage 1.3.6

  • 套件網站

NO.14 Download Statusbar 0.9.4

  • 套件網站

NO.15 Show Image 0.4.1

  • 原生中文

擴充套件是什麼玩意？

Firefox 有符合上網需求的基礎功能，若想加強它的能力，可安裝擴充套件。這些套件大多與 Firefox 完美整合，您只會感到 Firefox「長」出了一個功能，而不會當作使用另一套軟體。

其他資訊可參考社群成員路人乙撰寫的擴充套件 Q&A。
