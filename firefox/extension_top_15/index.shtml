第三屆 Top 15 擴充套件

在你心目中最棒的 Firefox 擴充套件是哪些呢？現在就來看看社群成員在 2008 年 8 月共同推舉的十五個最佳幫手！

想親自試試嗎？勾選套件的「列入安裝清單」後，按下頁尾的「安裝這些套件」便可一次安裝喔！您必須以 Firefox 觀看本頁才能安裝這些套件。

No.1 IE Tab

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「對 Firefox 不支援 IE only 的網頁感到非常不滿意？沒辦法，這不能怪 Firefox 啊！不過，試試 IE Tab 吧！」--社群成員 余弘兵

這是一個用來解決某些僅支援 IE 的網頁其瀏覽問題的解決方案，它讓您可以直接使用 IE 引擎瀏覽網頁，並能簡單的透過工具按鈕在 IE 與 Firefox 間切換。

適合 Windows 作業系統

No.2 新同文堂

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「看不懂對岸的字體？放心！不用隨身帶著繁簡對照字典，乾脆把責任交給繁簡轉換的老大哥：新同文堂吧！」--社群成員 余弘兵

可以轉換繁體或簡體的網頁，並有詞彙轉換表，不會有轉換問題出現。也有自動轉換功能能夠把任何繁體或簡體的網站自動轉換，還可以指定文字的字體，和把UTF-8等萬國碼的文字轉換。功能多到介紹不完，XD！（其實還有快速鍵、剪貼簿、文字縮放等功能。）

此版本無法自動更新，使用者需自行至套件網站尋找新版。

No.3 Adblock Plus

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「封殺網站上肆無忌憚的廣告。」--社群成員 浩

Adblock Plus 可以幫你擋掉你不想要看到的東西，例如圖片、Flash、Script、Frame 甚至 Popup。你可以自己設定要擋掉的網域跟內容。

No.4 Download Statusbar

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「從簡明的狀態列中檢視、管理下載的檔案，從此下載視窗不會再來阻擋你看網頁的視線了！」--社群成員 shyangs

Download Statusbar 可以在你下載檔案時，使用視窗底端的一行下載列取代 Firefox 的下載視窗。直覺、下載進度一目瞭然，也免去下載視窗的干擾。

No.5 ScrapBook

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「協助您擷取網頁，方便整理擷取後的資料。」--社群成員 浩

堪稱知識工作者必備套件，方便您擷取全部或部分網頁資料，並有方便的註解、標記、搜尋等功能。擷取下來的資料也可以離線閱讀或匯出供他人使用。

No.6 FireGestures

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「當初剛用 Firefox 3 beta 版時，All-in-one Gestures 並沒有支援 Firefox 3，於是找著找就找到這個套件，至此之後再也沒有跳回 All-in-one Gestures。」--社群成員 浩

易於自訂的滑鼠手勢套件，讓您可以使用五大類手勢、輕鬆執行各種命令。

No.7 Easy DragToGo

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「懶人必備。若有使用翻譯軟體，可以設定翻譯軟體的中鍵為翻譯熱鍵、連結開啟新分頁就交給滑鼠手勢。」--社群成員 vsy

簡單使用滑鼠手勢（上下左右），只要稍微拖曳目標就能辦到常用的動作，例如搜尋、開啟鏈結、另存圖片等等。

No.8 Foxmarks Bookmark Synchronizer

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「使用的電腦不只一台的人應該會很需要，像是公司和家中電腦的書籤同步、個人電腦與筆記型電腦中的書籤同步。另外同步的速度亦相當的快。」--社群成員 浩

如果你常在不同電腦上使用 Firefox，那麼一定要試試 Foxmarks。只要在每台電腦上安裝，Foxmarks 就會自動使每台電腦的書籤保持一致，還可以線上登入 my.foxmarks.com 管理你的書籤項目。

No.9 DownloadHelper

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「你還在為了抓取youtube的影片苦惱嗎？喜愛ニコニコ動画上的影片卻不知如何下手？再者聽到 yam 天空部落-影音分享的音樂想抓卻抓不到？這套件一次解決你的需求。」--社群成員 kikiqqp

從影音網站自動下載到你的電腦裡，還可設定自動轉檔。

No.10 Menu Editor

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「雖然功能就是這麼短短的一句話，但對我來說，它卻是不可或缺的套件。裝的套件一多，右鍵選單就會變的又臭又長，每次點了右鍵後還得要按右鍵選單下的箭頭才能找到我要的功能實在是會氣死人。」--社群成員 浩

Menu Editor 可以編輯您的右鍵選單及選單工具列，暫時移除您不需要的功能。

No.11 DownThemAll!

  • 套件網站
  • Mozilla 附加元件網站
  • 簡體中文
  • 另有正體中文版
列入安裝清單：
「可以掃描整個網頁裡的連結、嵌入圖檔，支援續傳、多線程分割下載檔案，與專業的下載工具 Flashget、Orbit 相比也毫不遜色。」--社群成員 shyangs

與 Firefox 整合的下載管理程式。

No.12 NoScript

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「網路上林林總總的網站上億個，當然也包括非常多不安全的網站跟網頁。一些不安全的程式語言混雜在網頁裡，讓人防不剩防，若是全部把 Firefox 的 JavaScript 功能關閉，似乎又會造成瀏覽上的困擾。此時 NoScript 就能派上用場。」--社群成員 goodjimmy

NoScript 可以針對網站開關 JavaScript、Java、Flash 等功能，防止 XSS 攻擊。待使用者確認網站可信任之後再加入白名單開啟。

適合進階使用者

No.13 Greasemonkey

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「這個套件很好很強大，連 Google 對其都畏懼三分。」--社群成員 shyangs

透過 Javascript，讓使用者自訂網站的功能與樣式。就算不會 Javascript，也可以去 Userscripts.org 找別人寫的 script 來安裝。

適合進階使用者

No.14 All-in-One Sidebar

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「可以把許多功能整合到側邊欄，滑鼠移到螢幕最左邊可以自動彈出，移離側邊欄會自動隱藏，我常用來點書籤；可以在側邊欄看頁面資訊不用另開視窗；開啟進階模式還可以把按鈕放到狀態列或分頁列...」--社群成員 shyangs

All-In-One Sidebar 是類似 Opera 的側邊列，點擊視窗左方即可切換各種側邊列。整合了好幾項貼心的小功能；除此之外，並特別支援數十個擴充套件。

No.15 gTranslate

  • 套件網站
  • Mozilla 附加元件網站
  • 原生中文
「有即時翻譯單字和整篇文章內容的功能；套件是抓取 Google 的翻譯功能，免除翻譯還要手動去開啟翻譯的網頁。」--社群成員 goodjimmy

gTranslate 能將選擇的文字、網頁送至 Google Translator 翻譯，將該網站的功能整合至 Firefox 中。

  • 如果您按下「安裝套件」卻無反應，請按下上方淡黃色訊息列之「允許」以便將 moztw.org 加入信任網站，再行按下「安裝套件」鈕。

擴充套件是什麼玩意？

Firefox 有符合上網需求的基礎功能，若想加強它的能力，可安裝擴充套件。這些套件大多與 Firefox 完美整合，您只會感到 Firefox「長」出了一個功能，而不會當作使用另一套軟體。

其他資訊可參考社群成員路人乙撰寫的擴充套件 Q&A。

這些套件裝了安全嗎？

MozTW 無法提供任何保證，若有顧慮建議您先參考隨附的連結：
  • 原文版或原生中文套件皆由套件作者提供，您可前往套件網站查閱相關資訊。
  • 「中文修改版」為社群成員製作，您可檢閱「另有中文版」的資訊。
您也可到擴充套件討論區或各修改版討論串發表意見，互助合作抓問題。
