Firefox 新鮮事

  • 全新功能 您可以通過鍵盤或耳機來暫停或撥放 Firefox 中的音訊或影片了，使您可以輕易的存取並控制您的媒體，即便正在其他分頁、程式、甚至在您的電腦鎖定時。
  • 全新功能 在原有的亮色與暗色佈景主題外，在這次的更新中 Firefox 加入了 Alpenglow 佈景主題：一個多彩的外觀，包含按鈕、選單、視窗。您可以在選項或偏好設定中更換您的 Firefox 佈景主題。
  • 全新功能 對於我們在美國與加拿大的使用者，Firefox 現在可以替您儲存、管理與自動填入信用卡資訊，在 Firefox 上購物將會更加便利。為了確保最穩定的使用體驗，這個功能將會逐步推出給使用者。
  • 全新功能 Firefox 支援 AcroForm，在之後讓您可以填入、列印與儲存支援的 PDF 表格，此外 PDF 閱讀器有了新的外觀。
  • 全新功能 位於澳大利亞、比利時、瑞士的德文版 Firefox 使用者現在會在新分頁看到 Pocket 推薦，精選網路上的奇文妙作。如果您沒看到這些內容的話，您可 參閱此步驟 打開新分頁的 Pocket 文章。除了 Firefox 的分頁，Pocket 也在 iOS 與 Android 上以應用程式的方式提供
  • 修正 修補幾種不同的安全弱點。
  • 修正 我們修復了語言包的使用者更新 Firefox 後會使預設語言重設成英文的問題。
  • 修正 瀏覽器原生的 HTML5 音訊／影片控制項獲得了數個重要的輔助功能修正：
    • 音訊／影片控制項仍然可以在螢幕閱讀器上被存取，即使在視覺上暫時性的被隱藏。
    • 音訊／影片已經撥放的時間與總時間現在可以在螢幕閱讀器上被存取，在之前的版本則無法。
    • 多個原本沒有被標記的控制項現在被標記了，使他們可以被螢幕閱讀器辨認。
    • 螢幕閱讀器不再侵入式的回報進度資訊，直到使用者要求。
  • 變更 在之後您可以更輕易的在觀看的影片中找到有新圖示的子母畫面（Picture-in-Picture）按鈕。
  • 變更 當匯入書籤到 Firefox 時，書籤工具列將會自動顯示，使您更容易地找到重要的網站。
  • 變更 我們加入了更多檔案的支援，包含 .xml、.svg 與 .webp，因此當您下載這些檔案後，可以直接在 Firefox 開啟。
  • 企業版 最新版本的 Firefox 加入了一些 Bug 修復與新的政策，您可以在 Firefox 企業版 81 Release Notes 獲得詳情。
  • 開發功能 TypeScript 現在能在除錯器中適當地被辨別並以相應的圖示標記，使您可以更容易的在檔案清單中找到。
  • 開發功能 使用 XSSI 防護字元的 HTTP JSON 現在會適當的解析，並以可展開的樹狀結構顯示 JSON 資料。使得這類 HTTP 回應可以透過傳統（可展開式）樹狀 UI 輕易的檢測。
  • 開發功能 現在暫停在腳本的第一個陳述式是可能的，這在某些狀況非常有用，像是開發者想對腳本執行或計時器產生的副作用除錯時。
  • 開發功能 網頁工具箱中位於輔助功能的色弱模擬現在更加準確。我們移除了紅色色弱、綠色色弱、藍色色弱的模擬，並且加入了全色盲模擬。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
