Firefox 新鮮事

  • 全新功能

    Disney+ 的子母畫面已經推出。

  • 全新功能

    Firefox 現在同時支援 scroll-snap-stop 以及re-snapping屬性。您可以使用 scroll-snap-stop 屬性的 always 和 normal 值來指定是否通過捕捉點，即使在快速滾動時也是如此。 在任何內容/佈局更改後，Re-snapping 會嘗試保持最後的捕捉位置。

  • 全新功能

    Firefox 分析器可以分析網站的電源使用率。(僅限於 Apple M1 及 Windows 11)

  • 全新功能

    Firefox UI 現在將於視窗最小化或是受到遮擋石限制效能及電池使用量，就如同背景分頁的效果一樣。

  • 修正

    在 Yahoo Mail 及 Outlook Mail 的編輯器當中按下 Enter 鍵之後，突出顯示顏色將正確保留下來。

  • 修正

    在繞過 HTTPS 的錯誤頁面之後，返回上一頁將帶你至先前已經關閉的錯誤頁面。現在會正確的帶你回到先前你所瀏覽的網站。

  • 修正

    黏貼無格式的內容（shift+ctrl/cmd+v）現在適用於純文字環境，例如輸入欄位和文字區域。

  • 修正

    修補多個安全弱點。

  • 變更

    透過將音訊解碼移動到更加嚴格的沙盒單獨處理程序中以提高安全性，從而改善程序隔離。

  • 企業版

    在最新的 Firefox 中修復多個 Bug 與加入新政策，您可在 Firefox 104 企業版版本資訊 之中找到更多資訊。
    Firefox 102 是新的擴展支援版本 （ESR）。Firefox 91 ESR 將於 2022 年 9 月 20 日停止支援。（有關詳細資訊，請參閱 102 ESR 發行說明）
