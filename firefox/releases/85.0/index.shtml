Firefox 新鮮事

  • 在 Mozilla，我們相信您有隱私權。您不該在上線時被追蹤，不論您正在查看銀行餘額、尋找最好的醫師或者買鞋子，不道德的追蹤公司不該有辦法在您瀏覽網路時追蹤您。因此，我們持續強化 Firefox，以替我們的使用者抵抗線上追蹤。
  • 全新功能 Firefox 現在能為您抵擋 supercookie，這是一種可以隱藏在您瀏覽器中並在線上追蹤您的追蹤器，即使您清除了 cookie。透過隔離 supercookie，Firefox 避免您在瀏覽不同網站時被追蹤。
  • 全新功能 儲存與取用您的書籤更為容易了。Firefox 現在會記得您偏好的書籤儲存位置、預設在新的分頁中顯示書籤列並讓您通過工作列資料夾取用所有您的書籤。
  • 全新功能 您現在可於密碼管理員中一鍵清除所有已儲存的登入資訊，而不必逐個刪除。
  • 修正 修補幾種不同的安全弱點。
  • 變更 Firefox 不再支援 Adobd Flash，已不再有重新啟用 Flash 支援的設定選項。
  • 企業版 多個 Bug 修復與新政策加入到最新的 Firefox 版本。您可在 Firefox 85 企業版版本資訊找到更多資訊。
  • 開發功能 CSS: 我們加入了對 :focus-visible 虛擬類別的支援。
  • 開發功能 在主控台中原始碼編輯器（多行編輯器模式下可用）可使用新的工具列按鈕來美化 JS 陳述式。

  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
