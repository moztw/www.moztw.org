Firefox 新鮮事

  • 全新功能

    我們已將搜索整合進 Firefox View。你現在可以搜索每個類別子頁面上的所有分頁——近期瀏覽、開啟的分頁、最近關閉的分頁、來自其他裝置的分頁、或瀏覽記錄。

  • 全新功能

    在 Firefox 上遇到網站問題，但該網站似乎在其他瀏覽器上運行良好？現在你可以通過「網路相容性工具」告訴我們！通過提交網路相容性問題，你可以直接幫助我們檢測、定位和修復影響最大的網站，讓你在 Firefox 上的瀏覽體驗更加順暢。（網址列的盾牌圖示 → 回報網站問題，或下拉選單 → 回報網站問題）

  • 修正

    翻譯網頁時，我們現在還會翻譯提示（tooltips）的 title 文字，以及表單控制項中顯示的提示文字（placeholder）。

  • 修正

    多項 安全性修正。

  • 變更

    現在可以在 Firefox 偏好設定的「搜索」中找到網址列相關設定。

  • 變更

    改進了 macOS ARM64 和 Android 的 PGO 優化。

  • 變更

    改進了 macOS、Linux 和 Android 的非主執行緒（off-main-thread）canvas。

  • 企業版

    你可以在 Firefox 123 企業版版本資訊 中找到政策更新和企業版相關的錯誤修復資訊。

  • 開發功能

    Firefox 123 開發者資訊

  • 開發功能

    網路監控器現在支援通過「另存回覆內容」右鍵選單將回覆內容（response body）儲存到硬碟。這也取代了僅支援保存圖片的「另存圖片為」右鍵選單。

  • 網路平台

    我們現在實作了 SVG 漸變的線性 RGB （linearRGB） 補插值，以補充現有的 sRGB 補插值。您可以通過 color-interpolation 屬性選擇使用哪一種插值。

  • 網路平台

    以前，指向 SVG 內容的 SVG feImage 元素要求 SVG 文件具有非百分比寬度和高度值的根元素。此類內容現在將回退到使用默認替換元素值的 300px 寬度和 150px 高度，因而 feImage 將不再顯示失敗。

  • 網路平台

    啓用預載入（Preload）和模塊預載（Modulepreload），現已完整支援 Early Hints。

  • 網路平台

    當聲音輸出被 setSinkId() 重定向到其他設備時，現在可以套用回音消除到麥克風輸入。

  • 網路平台

    Firefox 現在支持 Declarative ShadowDOM，為直接使用 HTML 中的 ShadowDOM 的開發人員提供更大的靈活度和工程面上的改善。

已知問題

  • 尚待修復

    一些具有較老的 AMD CPU 的機器，可能會在檔案對話框中看到圖片的縮圖全部呈現為不正確的黑色。如果發生這種情況，更新顯示卡驅動程式應該可以解決此問題。

  • 請參考此版本的完整變更清單。 你可能也想瞭解前一個版本的更新。
