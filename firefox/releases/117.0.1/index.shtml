Firefox 新鮮事

  • 修正

    在 macOS 上，修正從 Firefox 以外的地方開啟連結時無法開啟的問題 (bug 1850828)

  • 修正

    修正使用事件頁面來執行長時間任務的擴充套件，會在執行時被終止，而導致意外的行為變更 (bug 1851373)

  • 修正

    暫時還原了一項計畫中的行為變更，該變更會阻止 Javascript 更動 URL.protocol (bug 1850954)。
    附註：此項改變仍將會在稍後的 Firefox 版本中與其他網頁瀏覽器一同發布。如需變更協定時，我們鼓勵網站以其他方式進行。

  • 修正

    修正使用 WebAssembly 例外處理的網站 audio worklets 無法運作的問題 (bug 1851468)

  • 修正

    修正 最近關閉的分頁 選單中的 回復所有分頁 有時無法開啟所有分頁的問題 (bug 1850856)

  • 修正

    修正書籤選單有時在最小化 Firefox 時仍然部分可見的問題 (bug 1843700)

  • 修正

    修正某些網站偵測到的時區不正確的問題 (bug 1848615)

  • 修正

    安全性修正。
