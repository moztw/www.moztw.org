Firefox 新鮮事

  • 全新功能 效能提升：
    • 藉由 Quantum CSS 改良和 retained display list 功能加速頁面描繪。
    • 改善 Windows 和 Linux 中的分頁切換速度。
    • MacOS: 在獨立執行緒中運行 WebExtensions。
  • 全新功能 瀏覽提供 OpenSearch 功能的網站時，可以直接在網址列的「頁面操作」新增該搜尋引擎。
  • 全新功能 MacOS: 在網址列「頁面操作」選單中「分享」所在網址。
  • 全新功能 安全性改善：
    • 預設啟用最新 TLS 1.3 標準。
    • 封鎖 http(s) 頁面中對於 FTP 資源的存取。
  • 全新功能 暗色主題擴大涵蓋至整個 Firefox 介面。
  • 全新功能 附加元件支援「隱藏分頁」功能。
  • 全新功能 改善進書籤同步功能。
  • 全新功能 正體中文版: RSS 訂閱服務增加 Netvibes、Feedly、InoReader、The Old Reader（bug 918375）。
  • 修正 各種安全性修正。
  • 變更 首頁和新分頁的設定移至「選項」中的新區塊，可以由 about:preferences#home 或「新分頁」的齒輪圖示前往。

已知問題

  • 尚未修復 瀏覽安全網頁時，Avast 或 AVG 資安軟體的 HTTPS 掃描可能出現錯誤訊息（SSL_ERROR_RX_RECORD_TOO_LONG），在 Avast、AVG 修正前，可參考暫時性的解決方案。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
