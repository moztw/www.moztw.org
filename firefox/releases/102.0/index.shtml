Firefox 新鮮事

  • 全新功能

    厭倦了太多的視窗佔滿了你的桌面？從現在開始，你可以關閉在啟動新檔案下載時自動開啟下載面板。閱讀更多

  • 全新功能

    如果你啟用了隱私保護的嚴格模式，Firefox 現在可以減少你瀏覽網站時，網站連結自動帶入的追蹤參數。（如 fbcid 或 utm_*）

  • 修正

    修復在 Windows 系統上使用螢幕閱讀器時，按下 Enter 鍵後不會在觸發錯誤的網頁元素或是切換到其他應用程式的視窗。對於那些盲人或視力非常有限的人，這項技術可以大聲朗讀螢幕上的內容，使用者可以根據自己的需求進行調整。

  • 修正

    修補多個安全弱點。

  • 變更

    透過將音訊解碼移動到更加嚴格的沙盒單獨處理程序中以提高安全性，從而改善程序隔離。

  • 企業版

    在最新的 Firefox 中修復多個 Bug 與加入新政策，您可在 Firefox 102 企業版版本資訊 之中找到更多資訊。


    Firefox 102 是新的擴展支援版本（ESR）。Firefox 91 ESR 將於 2022 年 9 月 20 日停止支援。（有關詳細資訊，請參閱 102 ESR 發行說明）

  • 開發功能 Firefox 102 開發者資訊
  • 開發功能

    v現在你可以於開發人員工具的「樣式編輯器」中，選擇過濾樣式表。

  • 網路平台

    TransformStream 和 ReadableStream.pipeThrough 已經上架，允許你從 ReadableStream 通過通道傳輸到 WritableStream，並在每個區塊上執行轉換。

  • 網路平台

    ReadableStream、TransformStream 和 WritableStream 現在都可以被轉移了。

  • 網路平台

    Firefox 現在支援 WebAssembly 之中的 Content-Security-Policy （CSP） 集成。具有限制腳本的 CSP 的文件將不再被 WebAssembly 所執行，除非在策略中使用 'unsafe-eval' 或新的 'wasm-unsafe-eval' 關鍵詞。

  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
