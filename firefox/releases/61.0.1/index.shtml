Firefox 新鮮事

  • 參考 Firefox 新鮮事。
  • 修正 啟用 Windows 輕鬆存取功能的中國使用者無法載入網頁（bug 1471824）。
  • 修正 「新分頁」和選項中的「首頁」部分少了一些內容（bug 1471375）。
  • 修正 由 Firefox 60 升級時的書籤遺失問題（bug 1472127）。
  • 修正 改進 Twitch 1080p 影像串流播放功能（bug 1469257）。
  • 修正 顯示浮動提示時，焦點會離開所在的網頁（bug 1471415）。
  • 修正 Windows: 檔案不含副檔名時，無法開啟已下載的檔案（bug 1465458）。
  • 修正 恢復 HTTP 頁面中，對於 FTP 檔案的「鏈結另存新檔...」功能（bug 1470295）。
  • 修正 特定情況下，已停用的附加元件錯誤地覆蓋首頁設定（bug 1466846）。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
