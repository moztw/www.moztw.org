Firefox 新鮮事

  • 全新功能 隱私瀏覽「追蹤保護」可選用包含更多追蹤者的封鎖清單。
  • 全新功能 改善對 m4v 影片播放 API 的支援。
  • 全新功能 提供 Windows 版 64 位元 Firefox 下載。
  • 全新功能 可以選擇讓智慧網址列也提供搜尋建議。
  • 全新功能 選取文字欄位時，Windows 8 以上的裝置會顯示觸控式鍵盤。
  • 全新功能 Firefox 健康檢查報告使用和 Telemetry 相同的資料收集機制。
  • 開發功能 Markup 檢視新增鎖定 pseudo-class 的按鈕。
  • 開發功能 在開發工具中，按下 F1 會進入開發工具設定。
  • 開發功能 檢測器選單新增「在主控台中使用」，將元素儲存為便於操作的暫時變數。
  • 開發功能 規則面板新增「過濾包含此屬性的規則」按鈕在被覆蓋的 CSS 屬性旁，用來尋找相似的規則。
  • 開發功能 規則面板「過濾樣式」可以對屬性名稱進行搜尋。
  • 開發功能 在主控台顯示例外的堆疊追溯。
  • 開發功能 在主控台顯示伺服器端的 log 訊息。
  • 開發功能 可以選擇 GCLI 畫面擷取指令的解析度。
  • 開發功能 開發者可藉由子資源驗證打造更安全的網站。
  • 開發功能 點選主控台中的網路請求，將移動至「網路」面板而非打開新對話框。
  • 開發功能 支援 CSS hyphens 屬性，不需要瀏覽器 prefix。
  • 開發功能 WebIDE 改用以側邊欄為基礎的 UI。
  • 開發功能 CSS transform-origin 屬性在 SVG 元素也可使用。
  • 開發功能 檢測器「動畫」面板會以時間軸呈現動畫。
  • 開發功能 NPAPI 外掛程式不再支援 single-process 模式。
  • 修正 滴管工具在頁面縮放後運作不正確。
  • 修正 各種安全性修正。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
