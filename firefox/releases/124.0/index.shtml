Firefox 新鮮事

  • 全新功能

    PDF 檢視器內現在也可使用「鍵盤瀏覽」模式。(了解更多)

  • 全新功能

    在 Firefox View 中，「開啟的分頁」現在可以按「近期活動」或「分頁順序」排序，預設是依照近期活動排序。

  • 全新功能

    Firefox 現在以更有效率的方式填入 Windows 工作列的跳躍清單（jump list），應該可以帶來更流暢的整體瀏覽體驗。

  • 全新功能

    Mac 版 Firefox 現在在所有類型的全螢幕視窗上都使用 macOS fullscreen API，可在 macOS 全螢幕空間、選單列與 Dock 上提供更符合預期的使用體驗。

  • 全新功能

    從 Firefox 124 開始，Qwant 搜索引擎的可用範圍擴展至法國區域的所有語言，以及比利時、義大利、荷蘭、西班牙和瑞士等地。

  • 修正

    多項 安全性修正。

  • 企業版

    你可以在 Firefox 124 企業版版本資訊 中找到政策更新和企業版相關的錯誤修復資訊。

  • 開發功能

    Firefox 124 開發者資訊

  • 網路平台

    新增 AbortSignal.any 支援。

  • 網路平台

    建立 WebSockets 時你現在可以使用 HTTP(S) 與相對路徑（relative URLs）。

  • 請參考此版本的完整變更清單。 你可能也想瞭解前一個版本的更新。
