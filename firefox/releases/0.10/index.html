版本資訊 - Firefox 1.0 Preview Release ("Greenlane")

Firefox 是速度快、功能又齊全的瀏覽器，它讓上網比以往更有效率。其它關於 Firefox 的訊息，請參考 Firefox Help。

Firefox 1.0 Preview Release（以下簡稱為 1.0PR）為「技術預覽」版。即使本軟體性能良好，在大多數情形下足以成為您的主要瀏覽器，但我們不對其性能與可靠性做任何保證。這是尚未完工的版本，不應在重要關鍵作業上使用。詳情請見使用同意書。

此版本資訊包含了 Firefox 1.0PR 版的新鮮事、下載與安裝指示、已知問題以及常見問題集等訊息。如發現任何程式錯誤，請先行將版本資訊與錯誤回報注意事項閱讀完畢，再到 Bugzilla 回報。

我們想知道您對 Firefox 的意見。歡迎到討論區討論相關議題：您可至MozillaZine上的英文討論區或 MozillaTaiwan的Firefox 中文討論區與我們分享意見。

新鮮事

以下是 Firefox 此版的新鮮事：

  • Live Bookmarks

    您現在可以將 RSS feeds 訂閱成為書籤。當您參訪某個以 <link> 標籤公布 RSS feed 的網頁時，狀態列會出現 圖示。按下該圖示可查閱網頁提供的 RSS 檔列表。按下列表中的項目以便訂閱，如此便可新增一個書籤資料夾，其中含括此 RSS 檔的新近文章。

  • 增強版尋找功能

    有了新的 Find toolbar，尋找資料變得更加輕鬆且功能強大。Find toolbar（顯示於瀏覽器視窗底部）將自動標示您輸入的文字在網頁中的位置，並且也包含有用的標示（highlight）功能。

  • 掌控惱人事物，保護安全

    您現在可以開啟被阻檔的彈出視窗，此外擴充套件安裝系統會阻止 update.mozilla.org 之外的所有安裝程序。使用者可以新增軟體站台至允許清單中，但任何軟體絕不會自動安裝。除此之外也以多種方式避免 phishing attack，以安全式連線觀看網頁時也將加以標示。

  • 增強版書籤

    書籤系統有數項增強措施，包括更加確實的網站圖示及書籤視窗的分割檢視模式。

  • 提供高安全性密碼加密

    以密碼管理員儲存的密碼現在可以經由「主密碼」輕鬆加密。如果您建立了主密碼，每次只會要求您輸入主密碼一次，而後便能自動幫您輸入各網站的密碼。對多人共用電腦且希望提高安全性的人們來說是很有用的功能。

  • 提高與 IE 的相容性

    現在支援無從檢測的 document.all 以便提高網站相容性，也提高按鍵相容性以便讓 IE 的使用者轉換過程更加輕鬆。

  • 當然還修正了許多程式錯誤…

系統需求

在安裝 Firefox 前請先確定您的電腦合乎系統需求。

下載 Firefox

mozilla.org 提供了 Windows、Linux 及 Mac OS X 版的 Firefox。

Windows: 安裝程式

Linux: GTK2+ XFT 安裝程式

MacOS X: 已壓縮磁碟映像檔

中文化版本可至Mozilla Taiwan下載，此外尚有其他作業系統的非官方版本，請到英文版 Release Notes下載。

安裝 Firefox

下載 Firefox 安裝程式或壓縮檔後，請依下列步驟進行安裝：

Windows

以滑鼠左鍵雙點 Firefox-1.0PR.exe (中文版應為 /Firefox-1.0PR-win32-zhTW.exee) 安裝程式圖示以開始安裝。

MacOS X

以滑鼠雙點 Firefox 壓縮磁碟映像檔圖示以便裝載(mount)， 則瀏覽器便應能自動解壓縮並安裝完畢。再以滑鼠雙點解壓後的 Firefox 磁碟映像圖示，然後將 Firefox 應用程式拖曳到您的磁碟。如想讓圖示在 Dock 上顯示，可以自行將 Firefox 圖示拖曳至其上。

Linux/GTK2

解壓縮後以下列方式執行安裝程式：

tar -xzvf firefox-0.10-i686-linux-gtk2+xft.tar.gz
cd firefox-0.10-i686-linux-gtk2+xft/
./firefox-installer

其它平台

將壓縮檔解壓縮後執行 firefox

移除 Firefox

欲移除 Firefox，請依下列步驟操作：

Windows

選擇「開始＞控制台」，當控制台出現時，請以滑鼠左鍵雙點「新增／移除程式」， 然後在清單中找到「Mozilla Firefox」並按下「移除」。

MacOS X

將 Firefox 程式拖曳至垃圾桶。

其他平台

移除 firefox 資料夾。

上述步驟將保留您的個人設定檔，供日後重新安裝時使用。如果您想移除 Firefox 的個人設定檔資料夾，請依循以下步驟：

Windows

通常在 Documents and Settings\<使用者名稱> 的目錄下您能找到一個 Application Data 資料夾； 由於該資料夾預設是隱藏的，故請於檔案總管設定顯示隱藏檔。如果您的使用者名稱為「Joe」，且您的 Documents and Settings 資料夾位於 C:\ 之中，則您的個人設定檔便在C:\Documents and Settings\Joe\Application Data\Mozilla\Firefox 裡，將該 Firefox 資料夾刪除即可。

MacOS X

開啟您 Home folder 中的 Library 資料夾，並將其中的 Firefox 資料夾移除。 (~/Library/Firefox/)

其他平台

移除 ~/.firefox 資料夾。

您該知道的更動

個人設定檔資料夾

現在 Firefox 將您的個人設定檔存放在下列位置之一:

Windows Documents and Settings\<UserName>\Application Data\Mozilla\Firefox
MacOS X ~/Library/Application Support/Firefox
Linux ~/.mozilla/firefox/

如果您在升級至 Firefox 1.0PR 前將 Firefox 0.8 設為預設瀏覽器，那麼您的設定檔將被複製至新的位置， 您可自行決定是否將舊的「Phoenix」刪除。

舊版擴充套件及佈景主題

在 0.8 版時部分使用者曾因「No XBL Binding for Browser」錯誤訊息而無法啟動 Firefox，並隨即發現佈景主題失效。為了避免這種問題，在您改用 Firefox 1.0PR 時，所有的擴充套件將自動停用。如果您曾使用 0.9版，則當您首次啟動 1.0PR 時，Firefox 會試著連上 update.mozilla.org 查閱擴充套件是否有更新的相容版本。如果有，則會下載並安裝之。如果您用的是 0.8 版，則得請您自行尋找相容新版。

已知問題

以下列出 Firefox 1.0PR 的部分已知問題， 請閱讀後再回報新的程式錯誤，並請定期檢查本欄的更新內容﹝如在此版本發現新問題，我們會隨時在英文網頁更新此欄﹞。

  • 如果您將 Firefox 1.0PR 安裝於權限有所管制的區域，那麼就得先由擁有權限的使用者啟動一次 Firefox，以便建立應有的初始啟動檔案。若由權限不足的使用者先行啟動，則可能會因為無法完整建立初始啟動檔案而不斷重新啟動。這個問題將於日後修正。Mac OS X 使用者:不要由磁碟映像中啟動 Firefox！這麼一來 Firefox 可能會不斷重新啟動。若要從不斷啟動的情形中終止 Firefox，請開啟終端機（Terminal）後輸入「killall firefox-bin」並按下 Enter。
  • 若 Firefox 於 MacOS X中未能顯示瀏覽器視窗，請以Cmd+Q 跳出 Firefox，並開啟 ~/Library/Application Support/Firefox/Profiles/default.abc/且移除 localstore.rdf檔、重新啟動 Firefox。您之前的自訂的工具列及視窗位置等設定將因而消失。
  • Firefox 使用大量 Mozilla Application Suite 的程式碼， 許多您可能遇到的問題事實上是 Mozilla 核心程式的問題。如果您發現頁面顯示或連線的問題，它極可能是 Mozilla 的問題，也應被回報到 Bugzilla 的 Browser 產品類而非 Firefox。
  • 我們正小心翼翼的移往擴充套件自動更新的新天地，試圖以最快速度矯正任何問題。
  • 擴充套件及佈景主題的作者必須在每回 Firefox 推出新版本時更新 update.mozilla.org（或私有更新服務）的項目，以便讓我們知道是否相容。如果您發現您愛用的擴充套件及佈景主題於升級至 1.0PR 後無法使用，請寫信給作者並鼓勵他們更新或更動它們私有的升級服務檔。如果他們未將製品送上 update.mozilla.org，也請鼓勵他們送過來，這麼一來控制版本會容易許多。

欲取得其他問題的訊息、常見問題集、使用技巧以及一般 Firefox 協助，請到 David Tenser 非常有用的 Firefox Help網站、MozillaZine的Firefox 英文討論區或林弘德所維護的Firefox 中文討論區。

疑難排解

從舊版 Firefox 升級並不需建立新的設定檔，只要您在升級前先停用舊版中所有的擴充套件即可。您可以開啟舊版 Firefox，打開「工具＞選項」，然後選擇「擴充套件」，點選列表中每一個擴充套件，並且按下「停用擴充套件」以將其關閉。按下「確定」關閉選項視窗，便可放心安裝新版 Firefox 了。

Firefox 安裝完畢後，請檢查您使用的擴充套件是否更新以支援新版 Firefox，並下載安裝。未來這個動作將會自動處理。

如果您在舊版 Firebird 或 Phoenix 啟用某些擴充套件並發生了問題，請啟動舊版瀏覽器並停用擴充套件，再重新執行新版。

如果您遇到的怪問題與書籤、下載、視窗位置、工具列、瀏覽記錄或其他設定項目有關，建議您在回報問題前先行建立新的使用者設定檔，並查看能否重現該問題。建立新設定檔的方式如下：執行 Firefox 時加上 -P 的命令列參數，選擇「管理使用者設定檔」按鈕，然後選擇「建立使用者設定檔」。將各種設定(書籤、儲存的密碼等)從舊的設定檔中分次轉移到新的設定檔，每移轉一項便檢查是否發生問題。如果您發現某特定設定檔資料會造成問題，請回報錯誤並且附上該檔案。

常見問題集

  1. 我該怎麼幫忙呢？

    我們需要大量曝光。讓推廣 Mozilla Firefox 成為您的任務，使您的親朋好友上司下屬都認識它： 如果您是個學生，就讓它廣佈校園之中；寫篇文章投稿至Slashdot等新聞站台上，發布 Firefox 新版的新聞；也可以在您的 Blog 上透漏一點相關訊息；總之，儘可能的在網際網路中散佈，就像 Kazaa 的威力一樣，無所不在！

  2. 我使用其他瀏覽器，因為…

    如果您不願使用 Firefox，請告訴開發團隊您的理由。我們會持續注意Firefox forums中的使用者意見。

  3. 你們為什麼都沒有回信？

    請使用討論區，Firefox 團隊會定期閱讀上面的文章。我們常常都收到大量的信件，而您的可能不小心漏失了。

  4. 我該去哪裡取得新的佈景主題與擴充功能？

    擴充套件及佈景主題可自Mozilla Update下載。

  5. 哪些人正致力於 Firefox 的開發工作？

    很多人。詳細列表請參考「說明＞關於 Firefox」

  6. Firefox 的原始碼在何處？

    cvs.mozilla.org. Mozilla trunk + mozilla/browser + mozilla/toolkit.

  7. Firefox 是否只是可調整使用者介面的 Mozilla？

    Firefox 在本質上與 Mozilla 有極大差異，除了一些專有的特色之外也在程式上有難以計量的最佳化。自此專案開始後，瀏覽器及 toolkit 的 CVS 目錄中已更動或新增超過 120,000 行程式碼。

  8. 原來的名字「Phoenix」、「Firebird」又怎麼了？

    由於商標的問題我們必需更名， 新的名稱叫做「Firefox」。欲尋更多資訊，請見更名常見問題。

  9. 電子郵件程式呢？

    Firefox 能與您預設的任何郵件程式協同運作， 而我們推薦您試試我們下一代的郵件軟體：與 Firefox 合作無間的 Mozilla Thunderbird。
