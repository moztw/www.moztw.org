Firefox 新鮮事

  • 全新功能 藉 TSF（Text Services Framework）改善 Windows（Vista+）IME 支援。
  • 全新功能 可以為 Firefox 帳號設定個人圖片。
  • 全新功能 Firefox Hello 開始支援即時訊息。
  • 全新功能 網站 favicon 可以使用 SVG 圖片。
  • 全新功能 改善 box-shadow 繪圖效能。
  • 變更 WebRTC 強制要求 PFS（Perfect Forward Secrecy）支援。
  • 變更 停用 Windows 7 的 WARP。
  • 變更 關於圖片解碼程序的更新。
  • 變更 支援在 compositor thread 執行 CSS transform 和 opacity 動畫。
  • HTML5 啟用 MessageChannel 和 MessagePort API。
  • HTML5 支援 SVG 元素的 transform-origin 屬性。
  • HTML5 啟用 CSS Font Loading API。
  • HTML5 Navigator.onLine 會反映實際連線狀態（限 Windows 和 Mac OS X）。
  • HTML5 使用 JavaScript document.execCommand("cut"/"copy") 將網頁內容剪下／複製到 OS 剪貼簿。
  • HTML5 實作 Cache API，由 Window、Worker 或 ServiceWorker 存取快取資料。
  • 開發功能 擴充套件不再支援使用二進位 XPCOM 元件，請改用附加元件 SDK 的「system/child_process」pipe 機制。
  • 開發功能 網路請求可以使用 HAR 格式匯出。
  • 開發功能 使用檢測器的「新增規則」按鈕快速新增 CSS 規則。
  • 開發功能 使用標記檢視選單的「擷取該節點的圖片」對節點或元素擷圖。
  • 開發功能 使用檢測器選單的「複製屬性宣告」複製元素的 CSS 規則宣告。
  • 開發功能 檢測器新增「pseudo-class」面板。
  • 修正 Picture 元素不能正常反應 resize/viewport 變化。
  • 修正 各種安全性修正。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
