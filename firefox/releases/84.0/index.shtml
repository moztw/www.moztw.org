Firefox 新鮮事

  • 2020 是不同以往的一年。三月時，整個世界有著劇烈的變動。即便如此，我們仍按照計畫每個月發佈一版新的 Firefox 以讓 Firefox 可為您所用。雖然我們期待 2020 的結束，但我們想回顧這無法預期的一年。除了這一版本的新功能外，請參閱我們列出使 2020 好點的主要功能。
  • 全新功能 對 Apple Silicon CPU 提供原生支援，相比 Firefox 83 的非原生的建置帶來顯著的效能改善。Firefox 啟動快了 2.5 倍，網路應用程式現在的響應速度則快了兩倍（每個 SpeedoMeter 2.0 測試）。如果您使用新的 Apple 裝置，參考這些步驟來升級到最新的 Firefox。
  • 全新功能 推出 WebRender 給 MacOS Big Sur、搭載 Intel Gen 6 GPU 的 Windows 裝置、運行 Windows 7 與 8 的 Intel 筆記型電腦。此外，我們將首次為 Linux/GNOME/X11 的使用者提供加速渲染管道。
  • 全新功能 Firefox 現在使用更多現代技術以在 Linux 上分配共用記憶體，改善效能並提生與 Docker 的相容性。
  • 全新功能 Firefox 84 是最後一個支援 Adobe Flash 的版本。
  • 修正 修補幾種不同的安全弱點。
  • 企業版 多個 Bug 修復與新政策加入到最新的 Firefox 版本。您可在 Firefox 84 企業版版本資訊找到更多資訊。
  • 開發功能 網路面板現在可以處理非預期的當機並顯示有用的除錯資訊，像是相關的 Stack-trace。使用者可以簡單地透過點擊連結來送出 Bug 報告，幫助改善這個工具的穩定性。

  • 開發功能 輔助功能面板現在包含一個選項來顯示元素的 Tab 跳轉順序，以幫助開發者檢視哪些元素是可透過 Tab 按鍵聚焦與其順序。
  • 尚未修復 運行在 Apple Silicon 系統上的 macOS 使用者如果沒有安裝 Rosetta 系統軟體，在播放加密內容時可能遇到播放錯誤。
  • 尚未修復 運行 Cylance 防毒軟體的 macOS 使用者可能會看到 Firefox 安裝過程被誤判為惡意軟體而中斷。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
