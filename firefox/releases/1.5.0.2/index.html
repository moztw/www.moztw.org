Firefox 1.5.0.2 新鮮事

Firefox 1.5.0.2 現在原生支援採用 Intel Core 處理器的麥金塔電腦，和數項穩定性及安全性修正。此次更新是我們提供給顧客更安全的網路體驗計畫的一部份。建議所有使用者都更新到最新版本。

Firefox 1.5.0.2 版中較值得一提的有：

  • 提供 Mac OS X 的 Universal Binary 支援，在採用 Intel Core 處理器的麥金塔電腦上不需透過 Rosetta 便可執行。
  • 增進產品穩定性。
  • * 數項安全性修正。

* Burning Edge 有更詳細的錯誤修正列表。

* 此為英文網站

下載與安裝

系統需求

在安裝 Firefox 前請先確定您的電腦合乎系統需求。從 1.5 開始，MacOS X 用戶必須在 Mac OS X 10.2 以上版本執行本程式。

下載 Firefox

Mozilla.org 提供了 Windows、Linux 及 Mac OS X 平台上多種語言版的 Firefox。欲下載最新的 Firefox，可於產品首頁或到 FTP 下載。若想自行編譯其他不支援的平台或語言，請見本文件最末節。

安裝 Firefox

下載 Firefox 安裝程式或壓縮檔後，請依下列步驟進行安裝：

Windows

以滑鼠左鍵雙點 Firefox Setup 1.5.0.2.exe 安裝程式圖示以開始安裝。

MacOS X

下載 Firefox 1.5.0.2.dmg 檔後，請雙點 Firefox 磁碟映像檔圖示以便開啟於 Finder 中，隨後將 Firefox 應用程式拖曳至您的硬碟中。請不要直接雙點磁碟映像檔中的圖示！一定要將 Firefox 相關檔案拖進硬碟後才執行。您亦可將圖示拖曳至 Dock 上方便取用。

Linux/GTK2

請將下載回來的 tarball 檔解壓縮至您想要安裝 Firefox 的資料夾：

tar -xzvf firefox-1.5.0.2.tar.gz

這會自動建立 firefox 子目錄，之後執行 firefox 即可。

其它平台

將壓縮檔解壓縮後執行 firefox。

移除 Firefox

欲移除 Firefox，請依下列步驟操作：

Windows

選擇「開始＞控制台」，當控制台出現時，請以滑鼠左鍵雙點「新增或移除程式」， 然後在清單中找到「Mozilla Firefox (1.5.0.2)」並按下「移除」。

MacOS X

將 Firefox 程式拖曳至垃圾桶。

其他平台

移除 firefox 資料夾。

上述步驟將保留您的個人設定檔，供日後重新安裝時使用。如果您想移除 Firefox 的個人設定檔資料夾，請依循「個人設定資料夾」一節的步驟。請注意：這麼一來您將失去所有書籤、儲存密碼、設定及其他資訊，重新安裝 Firefox 時就必須重新建立個人設定。

 

其他資訊

個人設定資料夾

Firefox 將您的個人設定檔存放在下列位置之一：

Windows 2000, XP Documents and Settings\<UserName>\Application Data\Mozilla\Firefox
Windows NT WINNT\Profiles\<UserName>\Application Data\Mozilla\Firefox
Windows 98, ME Windows\Application Data\Mozilla\Firefox
MacOS X ~/Library/Application Support/Firefox
Linux 及 Unix 系統 ~/.mozilla/firefox

如果您在升級至 Firefox 1.5 前將 Firefox 0.8 設為預設瀏覽器，那麼您的設定檔將被複製至新的位置， 您可自行決定是否將舊的「Phoenix」資料夾刪除。

其他變動

  • Firefox 現在於 Linux 上遵照 GTK 於文字輸入欄的鍵盤設定，所以 Ctrl+字母 為程式快速鍵而非 readline-like text-editing shortcuts. 請參考 這份文件以獲得其他資訊。

 

已知問題

以下列出 Firefox 1.5.0.2 部分已知問題，請閱讀後再回報新的程式錯誤，並請定期檢查本欄的更新內容（如在此版本發現新問題，我們會隨時在英文網頁更新此欄）。

擴充套件與佈景主題

您可能會在我們新設計的擴充套件相容和更新機制上遇到問題。下面列出了已知三個問題的簡單解決方法。

  • 當您執行 Firefox 安裝程式後，沒有出現擴充套件相容性和更新精靈的話，您可以打開「擴充套件管理員」，手動檢查更新檔。
  • 如果擴充套件相容性和更新精靈下載並安裝了更新檔，但是安裝失敗的話，請打開「擴充套件管理員」再次檢查有無更新檔。
  • 在 Intel 麥金塔電腦上，原本設計僅限 PowerPC 的套件可能無法正確運作，除非您將套件升級到支援 Intel 或 Universal 的版本，或是開啟 Rosetta。要開啟 Rosetta，請點擊 Firefox 的圖示，按一下「取得資訊」後再勾選「於 Rosetta 下執行」。

在 Firefox 1.0 安裝的擴充套件，到了 Firefox 1.5.0.2 時可能並不相容。這時您需要檢查該套件有無更新檔，使其可在 Firefox 1.5 上運作。如果您喜歡的擴充套件或佈景主題未能相容新版，請寫信給作者，鼓勵作者更新。當您升級到 Firefox 1.5.0.2 時，所有的擴充套件及佈景主題會自動停用，直到 Firefox 發現 1) 此套件原本就相容於 Firefox 1.5.0.2 或 2) 有新版套件可相容於 Firefox 1.5.0.2。這是因為 Firefox 每一版都有所更新，與擴充套件的整合模式也不盡相同，如果使用舊版套件反而可能發生問題。

所有系統

  • 某些軟體防火牆執行時會悄悄封鎖 Firefox 的對外連線，但大部分的防火牆會先問您是否允許此程式連線到網路上。要能正常執行 Firefox，請確定您選擇了「永遠允許」Firefox 的網路存取。如果您選擇拒絕連線，您必須到防火牆的組態設定中重新允許連線。某些軟體防火牆就連停用時還是會持續封鎖網路連線。此篇文件詳細說明了如何設定您的防火牆。
  • 如果您將 Firefox 安裝於權限有所管制的區域，那麼就得先由擁有權限的使用者啟動一次 Firefox，以便建立應有的初始啟動檔案。若由權限不足的使用者先行啟動，則可能會因為無法完整建立初始啟動檔案而不斷重新啟動。如果問題還是無法解決，請於下列路徑建立一空白檔案：<install-directory>/extensions/talkback@mozilla.org/chrome.manifest
  • 若您未擁有 Firefox 安裝處的檔案寫入權限，軟體升級程序便無法進行，這是因為軟體升級必須於安裝處替換或建立檔案。
  • 若您於 Firefox 1.0 版本的設定中未勾選「允許網站安裝軟體」的選項而使 xpinstall 停用。（此設定於 Firefox 1.5 以上已移除），請用 about: config 將 xpinstall.enabled 設定為 true 即可重新啟用安裝功能。
  • 較舊的 Adblock 版本可能會使 Firefox 啟動時自動出現「清除隱私資料」視窗。如果妳遇到此問題，只要將您的 Adblock 或 Adblock Plus 升級到最新版本即可。

Windows

  • 以權力受限的一般使用者身份安裝 Firefox 時，就算安裝在可寫入的地方，有時依然會發生些許問題（像是預設瀏覽器設定錯誤等），但瀏覽器還是可以用。當以受限使用者身份安裝時，請不要在未移除舊版的情況下指定安裝於同一資料夾，否則可能無效。
  • 若使用某些舊版的 Adobe Acrobat Reader plug in，閱讀 PDF 後 Firefox 可能會當掉。如果您碰到這種問題，請到 Adobe 網站下載 新版 Plug in。

Mac OS X

  • 如果您現正透過 Rosetta 於 Intel Core 的麥金塔電腦上執行 Firefox，則透過自動更新系統所取得的還是給 PowerPC 的版本。您必須從 www.mozilla.com 重新下載完整的 Universal Binary 版才可不透過 Rosetta 在 Intel Core 的麥金塔電腦上原生執行 Firefox。
  • Java 不能於 Rosetta 下執行。
  • 在 Intel Core 的麥金塔電腦上執行時，不論原生執行或是透過 Rosetta 執行，都沒有 Talkback（錯誤回報）功能。當錯誤發生時，Apple Crash report 程式會自動執行。

Linux 及 Unix 系統

  • 若 Firefox 安裝的路徑中含有空白字元，則可能無法設定為預設瀏覽器。這樣每次啟動時變會不斷詢問「是否要設為預設瀏覽器」。將 Firefox 安裝在不含空白字元的路徑就不會有這個問題。
  • 在 Fedora Core 3(FC3) 中，GNOME integration 並不能適當的運作。 FC3 的使用者需要安裝 linc-1.0.3-3.1.i386.rpm。安裝好這個 RPM 後，在您安裝 Firefox 的目錄下執行以下的命令(您需要有寫入的權限)：

    touch .autoreg

    下一次您啟動 Firefox 時，GNOME integration 將會正常的運作。

  • 以 firefox -remote (mozilla-xremote-client) 指令啟動時，若參數網址包含逗號將不會有效，且不會修正。請使用 firefox -new-window /url/ 會在新視窗開啟網址；firefox -new-tab /url/ 則會在新分頁開啟網址。

網頁展現問題

  • Firefox 與其他 Mozilla 產品都使用相同的 Gecko 佈局引擎。如果您發現有某個網站無法正確顯示，那大多屬於 Gecko 的問題，而與 Firefox 本身無甚牽連。這些問題應於 Bugzilla 中提報為 Browser 產品類錯誤，而非提報為 Firefox 錯誤。如果您對技術小有心得，請試著找出引發問題的原因，並製作測試網頁以便更精確描述問題所在。

欲閱讀其他問題的訊息、常見問題集、使用技巧以及一般 Firefox 協助，請參考 Firefox Help 及 MozillaZine 的 Firefox 英文討論區。中文相關資訊則可參考社群成員維護的 Firefox FAQ或Firefox 中文討論區。

設定控制台（在網址列輸入「about:config」後按 Enter 時出現的東西）提供 Firefox 進階操作控制，可以直接修改 Firefox 設定。這些數值若未善加修改，可能會毀了 Firefox！請再修改前閱讀說明文件，以便得知相關數值如何調整。

疑難排解

  • 設計不良或不相容的套件將導致瀏覽器發生問題，發生的問題包含當機或網頁瀏覽速度變慢等等。如果您遇到瀏覽器無法開啟、視窗顯示不正常等等奇怪的問題，您可能是遇到擴充套件或佈景主題的問題。將瀏覽器重新啟動在安全模式。在Windows下用「Safe Mode」捷徑或在命令列下firefox.exe -safe-mode指令。在Linux下執行./firefox -safe-mode指令。在MacOS X下

    cd /Applications/Firefox.app/Contents/MacOS/
    ./firefox-bin -safe-mode

    當啟動安全模式時，所有的擴充套件將停止使用，並使用預設的佈景主題。在安全模式下移除有問題的擴充套件/佈景主題，然後再正常的啟動瀏覽器。

如果您想在移除某擴充套件後，以 -install-global-extension 指令將同一擴充套件裝到每個使用者設定檔中，則必須在重新安裝前先重開瀏覽器一次，以便徹底移除擴充套件。如果您沒這麼做，則擴充套件清單可能會有衝突項目而不能為所有設定檔安裝。

如果您遇到的怪問題與書籤、下載、視窗位置、工具列、瀏覽記錄或其他設定項目有關，建議您在回報問題前先行建立新的使用者設定檔，並查看能否重現該問題。建立新設定檔的方式如下：執行 Firefox 時加上 -P 的命令列參數，選擇「管理使用者設定檔」按鈕，然後選擇「建立使用者設定檔」。將各種設定(書籤、儲存的密碼等)從舊的設定檔中分次轉移到新的設定檔，每移轉一項便檢查是否發生問題。如果您發現某特定設定檔資料會造成問題，請回報錯誤並且附上該檔案。

 

常見問題集

  1. 我該怎麼幫忙呢？

    我們需要大量曝光。讓推廣 Mozilla Firefox 成為您的任務，使您的親朋好友上司下屬都認識它： 如果您是個學生，就讓它廣佈校園之中；寫篇文章投稿至Slashdot等新聞站台上，發布 Firefox 新版的新聞；也可以在您的 Blog 上透漏一點相關訊息； 總之，儘可能的在網際網路中散佈，就像 Kazaa 的威力一樣，處處推廣 Firefox！

  2. 我使用其他瀏覽器，因為…

    如果您不願使用 Firefox，請告訴開發團隊您的理由。我們會持續注意Firefox forums中的使用者意見。

  3. 你們為什麼都沒有回信？

    請使用討論區， Firefox 團隊會定期閱讀上面的文章。我們常常都收到大量的信件，而您的可能不小心漏失了。

  4. 我該去哪裡取得新的佈景主題與擴充功能？

    擴充套件及佈景主題可自Mozilla Update下載。

  5. 哪些人正致力於 Firefox 的開發工作？

    很多人。詳細列表請參考「說明＞關於 Firefox」

  6. Firefox 的原始碼在何處？

    Firefox 1.5 原始碼可點此下載，目前最新的程式碼則需以 CVS 取得。Firefox 相關程式為「mozilla/browser」、「mozilla/toolkit」及「mozilla/chrome」，請參考建製步驟。

  7. 電子郵件程式呢？

    Firefox 能與您預設的任何郵件程式協同運作， 而我們推薦您試試我們下一代的郵件軟體：與 Firefox 合作無間的 Mozilla Thunderbird。

  8. Firefox 要怎麼拼？簡寫又是什麼？

    就拼做 F-i-r-e-f-o-x，只有第一個字是大寫，不是 FireFox、Foxfire、FoxFire 等玩意。我們偏好的簡寫方式為「Fx」或「fx」。
