Firefox 新鮮事

  • 全新功能 強化追蹤防護：內容封鎖設定簡化為「標準」、「嚴格」與「自訂」三種選項，「網站資訊」面板（在網址列的 i 圖示）內容追蹤區塊也做了重新設計，顯示 Firefox 在各網站偵測到的內容。詳情參考 Mozilla Blog。
  • 全新功能 改善多語系使用者體驗：語言選項中的「設定其他語言」允許安裝額外語言供 Firefox 介面、網頁使用，而不必安裝特定語言版本的 Firefox。
  • 全新功能 macOS: 支援 Handoff，在 iOS 瀏覽到一半的網頁（使用 Firefox 或 Safari），可以在 Mac Firefox 上繼續。
  • 全新功能 Windows 64-bit: 支援次世次免費影片壓縮技術 AV1，參考 Mozilla 為此新開放標準進行的努力。
  • 全新功能 改進效能與 web 相容性，支援 WebP 圖片格式，帶來同樣的影像品質，只需較小檔案大小。
  • 修正 各種安全性修正。
  • 變更 macOS, Linux 和 Android 上的安全性改進：預設啟用較強的 stack smashing 防護。
  • 變更 無論是否設定在重啟時自動還原瀏覽階段，關閉視窗時都會發出警告。
  • 變更 簡化效能管理：由 about:performance 進入重新設計的工作管理員頁面，檢視各分頁、附加元件的記憶體用量。
  • 變更 改善封鎖視窗功能，防止網頁同時開啟多個彈出視窗。
  • 企業版 Windows: 提供 32 和 64 位元版的 MSI 安裝檔。
  • 開發功能 改善 Flexbox 支援：新的 Flexbox 檢測器能檢視 Flexbox 容器細節以利除錯。
  • 開發功能 所有在「規則」面板變動的 CSS 都能在新的「變更」面板中追溯。
  • 開發功能 於各桌面版平台皆支援 Storage Access API。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
