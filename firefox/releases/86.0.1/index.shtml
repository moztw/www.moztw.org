Firefox 新鮮事

  • 在 Mozilla，我們相信您有隱私權。您不該在上線時被追蹤，不論您正在查看銀行餘額、尋找最好的醫師或者買鞋子，不道德的追蹤公司不該有辦法在您瀏覽網路時追蹤您。因此，我們持續強化 Firefox，以替我們的使用者抵抗線上追蹤。
  • 修正 修正了 Apple Silicon 晶片裝置上導致 Firefox 在系統休眠之後沒有反應的問題。 (bug 1682713)
  • 修正 修正了視窗意外獲取或失去焦點的問題。 (bug 1694927)
  • 修正 修正寬度運算錯誤導致日期和時間選擇器 (date and time widget) 被裁切的問題。 (bug 1695578)
  • 修正 修正導致分頁管理擴充套件行為異常的問題。 (bug 1694699)
  • 修正 修正 Linux 開啟瀏覽器時時常 crash 的問題。 (bug 1694670)
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
