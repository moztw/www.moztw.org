Firefox 新鮮事

  • 全新功能

    右鍵選單中新增「複製不含追蹤碼的鏈結」功能，確保複製的連結不包含追蹤資訊。

  • 全新功能

    Firefox 「偏好設定 → 隱私權與安全性」中新增 全球隱私控制 (Global Privacy Control, GPC) 選項。透過這項自願勾選的功能，Firefox 會告知網站「使用者不希望其資料被共享或出售」。此功能在隱私瀏覽模式下預設啟用。

  • 全新功能

    Firefox 的隱私視窗和「加強型追蹤保護」的「嚴格」模式，現在強化了 Canvas API 的指紋保護，以持續保護我們使用者的線上隱私。

  • 全新功能

    在接下來的幾週內，Firefox 將會在德國使用者的隱私視窗中預設啟動「Cookie 橫幅阻擋器」（Cookie Banner Blocker）。Firefox 現在將自動拒絕 cookie，並關閉支援網站上討人厭的 cookie 橫幅。

  • 全新功能

    現在德國所有使用者的隱私視窗中預設啟用 URL 追蹤保護。Firefox 將移除經常用來追蹤使用者的非必要 URL 查詢參數。

  • 全新功能

    Firefox 現在會從作業系統的根憑證庫匯入 TLS 信任錨點（例如憑證）。此功能在 Windows、macOS 和 Android 上預設啟用，如有需要的話，可在設定中關閉（偏好設定 → 隱私權與安全性 → 憑證）。

  • 全新功能

    about:logins 中新增了編輯和刪除選定憑證的鍵盤快速鍵。編輯為 Alt + enter（macOS 上為 Option + return），刪除為 Alt + Backspace（macOS 上為 Option + Delete）。

  • 全新功能

    在 Ubuntu Linux 上，當 Chromium 和 Firefox 皆透過 Snap 安裝時，即可從 Chromium 匯入資料。

  • 全新功能

    子母畫面在 Windows 和 Linux 上支援角落吸附功能－只需在移動子母畫面時按住 Ctrl 鍵。

  • 修正

    多項 安全性修正。

  • 企業版

    你可以在 Firefox 120 企業版版本資訊 中找到政策更新和企業版相關的錯誤修復資訊。

  • 開發功能

    Firefox 120 開發者資訊

  • 開發功能

    新增 User Activation API，使 JavaScript 可透過 navigator.userActivation 檢查使用者目前或是否曾經與頁面產生互動（如點擊等）。

  • 開發功能

    啟用「Early Hints Preconnect」（預先連線的提前提示），支援提供資訊的 103 Early Hints 狀態碼，使伺服器能夠在最終 HTTP 回應之前先行傳送含所需資源連結的檔頭，為使用此功能的伺服器改善效能。（詳細資訊）

  • 開發功能

    使用者現在可以透過新增的開發者工具，來模擬瀏覽器分頁處於離線狀態。（網路 → 不限速 → Offline）

  • 開發功能

    樣式編輯器面板底部現新增了一個「Pretty print 樣式表」按鈕，類似於除錯器面板中的 Pretty Print 原始碼按鈕。可用於格式化樣式表（例如哪些被壓縮過的樣式表）。先前自動格式化壓縮文件的功能已移除。

  • 開發功能

    檢測器面板中的規則面板，不再將新的 CSS Color 4 顏色（例如 OKLCH）顯示為十六進位／命名顏色，以確保它與原來使用的值相符。

  • 網路平台

    現在可以正確解析和計算 lh 及 rlh 單位為 長度，讓作者能根據元素（或根元素）的行高來指定長度。

  • 網路平台

    現已預設啟用 WebAssembly GC，允許新的語言──如 Dart 或 Kotlin──在 Firefox 上運行。這使得客戶語言和主瀏覽器之間的參考循環可以被回收。

已知問題

  • 尚待修復

    我們已注意到某些分頁大小非 4KB 的 aarch64 系統（如 Apple Silicon）上的 Linux 使用者的啟動時當機問題。為繞過此問題，請使用 --disable-phc 參數編譯 Firefox。
