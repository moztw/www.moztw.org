Firefox 新鮮事

  • 全新功能 針對 Windows 的效能與視覺改進：
    • 改用 Clang 工具鏈建置 Windows 版 Firefox，帶來可觀的效能提升。
    • 和 Windows 10 OS Dark / Light 模式更契合的主題。
  • 全新功能 針對 macOS 的效能改進：
    • 響應性提升。
    • 更快的分頁切換。
    • 在多 GPU 系統下，WebGL 電源選項可允許效能要求較低的應用程式、applet 優先選用低速 GPU。
  • 全新功能 新增「內容封鎖」設定，讓使用者掌握上網時的追蹤、監控行為。使用者可選擇啟用第三方 cookie 或阻擋所有追蹤器，同時自訂例外名單，讓需要追蹤的網站正常運作。
  • 全新功能 Linux: WebExtensions 會在獨立程序中執行。
  • 全新功能 由主選單離開 Firefox 時，若已開啟多個視窗／分頁則會顯示警告。
  • 全新功能 辨識作業系統的親和力功能，減少不必要的動畫效果。
  • 全新功能 Firefox 首頁（新分頁）搜尋捷徑：「熱門網站」會出現 Amazon 和 Google，點選後會移至網址列以便搜尋（目前僅 US 版本有效）。
  • 修正 特定情況下，網址列無法自動完成書籤網址。
  • 變更 收藏庫：移除各別書籤「在側邊欄開啟」功能。
  • 變更 移除「不要檢查更新」選項，您可使用 DisableAppUpdate 政策替代。
  • 變更 Ctrl + Tab 快速鍵現在會顯示各分頁縮圖，並依最近使用的順序切換。僅新的使用者會套用此行為，舊使用者可自行調整設定。
  • 開發功能 更新開發者工具選單外觀，改善導覽功能與一致性。
  • 開發功能 預設啟用輔助功能檢測器，幫助了解頁面上不足或應留意的輔助功能。
  • 開發功能 支援 Web Components 自訂元素和 shadow DOM。
  • 開發功能 檢測器增加字型編輯器，方便控制一般、可變字型。
  • 開發功能 參考開發者功能改進。
