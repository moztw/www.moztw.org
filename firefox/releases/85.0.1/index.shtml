Firefox 新鮮事

  • 修正 修復安全弱點。
  • 修正 禁止訪問可能導致文件系統損壞的 NTFS 特殊路徑。
  • 修正 修復了在具有 Apple Silicon CPU 的 macOS 設備上使用 SPNEGO 對網站進行身份驗證時崩潰的問題 (bug 1685427)。
  • 修正 避免在某些文檔的末尾打印多餘的空白頁 (bug 1689789)。
  • 修正 修復了意外的 Cache API 狀態時瀏覽器崩潰的問題 (bug 1684838)。
  • 修正 修復了使用 Firefox flatpak 時的外部 URL 方案處理程序 (bug 1688966)。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
