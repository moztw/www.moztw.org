Firefox 新鮮事

  • 全新功能 改善 JavaScript JIT (Just In Time) 編譯器安全性。
  • 全新功能 GTK3 整合（限 GNU/Linux）。
  • 修正 正確處理使用 clip 和 mask 的 SVG 縮放繪圖。
  • 修正 修正 Google Docs 螢幕閱讀器處理空白的行為。
  • 修正 各種安全性修正。
  • 變更 改善 WebRTC 效能和穩定性。
  • 開發功能 記憶體工具新增 dominator 顯示。
  • 開發功能 效能面板可以剖析記憶體配置和 garbage collection 暫停的清況。
  • 開發功能 樣式編輯器 @media 側邊欄可配合適應性設計檢視模式縮放畫面。
  • HTML5 支援 document.elementsFromPoint。
  • HTML5 Web Crypto API 新增 HKDF 支援。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
