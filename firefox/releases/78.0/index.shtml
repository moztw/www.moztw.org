Firefox 新鮮事

  • 全新功能 保護資訊儀錶板提供包含追蹤保護、資料洩漏與密碼儲存的綜合報告。新的功能提供了：
    • 從儀錶板追蹤多少洩漏事件已被解決
    • 檢查有沒有儲存的密碼因洩漏事件而暴露
    若要檢視你的儀錶板，在網址列輸入 about:protections，或者從主選單中選取「保護資訊儀錶板」。
  • 全新功能 我們發現很多人會為了解決某些只需重設就可解決的問題而重裝整個 Firefox，所以在安裝工具中加入了重設按鈕。
  • 全新功能 畫面擷圖工具不再會打斷 Firefox 上的 WebRTC 通話了，讓 Firefox 上的會議與影像通話體驗更好。
  • 全新功能 我們推出了 WebRender 給使用 Intel GPU 的 Windows 使用者們，將更好的圖形效能提供給更多人。
  • 全新功能 Firefox 78 也是我們長期支援版本（ESR），前十個更新的功能將開始提供給 ESR 的使用者。主要有：
    • Kiosk 模式
    • 客戶端憑證
    • 啟用 Service Worker 與 Push API
    • 啟用封鎖自動播放
    • 支援子母畫面
    • 在 about:certificate 中檢視與管理 Web 憑證
  • 全新功能 展示網路上最佳內容的 Pocket 推薦將會在所有英國使用者的新分頁中顯示。如果你沒有看到，你可以在新分頁中開啟 Pocket 文章，請參閱教學。
  • 修正 各種安全性修正。
  • 修正 修復了搜尋結果品質組成，並且改進了合作夥伴提供的搜尋建議文字。
  • 變更 Linux 的最低系統需求更新。現在 Firefox 需要 GNU GNU libc 2.17、libstdc++ 4.8.1 與 GTK+ 3.14 或更新。
  • 變更 由於我們正在努力廢止過時的加密法，所有現存的 DHE-based TLS 加密套件將預設停用。
    • 為了緩和因停用 DHE-based TLS 加密套件帶來的 Web 相容性問題，Firefox 78 啟用了額外兩個 AES-GCM SHA2-based 加密套件
  • 變更 停用 TLS 1.0 與 TLS 1.1 以優化網站連線。不支援 TLS 1.2 的網站將會顯示錯誤頁面。
  • 變更 環境選單（通過右鍵點擊分頁開啟）可以復原關閉多個分頁的操握，「關閉右方分頁」與「關閉其它分頁」被移到子選單中。
  • 變更 優化了幾個無障礙體驗：
    • 當使用 JAWS 螢幕閱讀器時，點選具有資料清單的 HTML 輸入控制項中的下箭頭不再錯誤地移動游標到輸入控制項的下一個元素。
    • 在螢幕閱讀器上聚焦到麥克風、相機與螢幕分享指示器時，不再導致嚴重的 Lag 或停止回應。
    • 大幅提高了有數千行的表格對於螢幕閱讀器的使用者的載入速度。
    • 文字輸入控制項在有自訂樣式時，能夠適當的顯示聚焦輪廓。
    • 在螢幕閱讀器開啟主要的開發者工具視窗時，不再會錯誤的切換成文件瀏覽模式。
    • 減少像是懸停分頁、展開搜尋列之類的動畫，以降低對患有頭痛與癲癇使用者的視覺刺激。
  • 企業版 將實驗性偏好設定 security.osclientcerts.autoload 設成 true ，即可啟用儲存在 macOS 或 Windows 的客戶端憑證的支援。
  • 企業版 提供新的政策以設置應用程式處理器、停用子母畫面與要求主控密碼（在未來的更新將從 Master password 更名為 Primary password）。
  • 企業版 詳細資訊請參閱企業版 Firefox 78 版本資訊。
  • 開發功能 開發工具箱主控台現在會對未捕捉的 Promise 錯誤紀錄更詳細的名稱、堆疊與屬性，特別是對 JavaScript 框架的除錯。
  • 開發功能 除錯器中最小化過的變數的自動對應現在可以在 LogPoints 中使用，讓 Source-mapped 專案的除錯者感到更流暢。
  • 開發功能 開發工具箱的網路面板將標記阻擋請求的插件或 CORS 限制，開發者將能讓網站更加彈性與安全。
  • 開發功能 SpiderMonkey 使用新的 RegExp 引擎，加入了 dotAll 標誌、Unicode 跳脫序列、向後參考與命名擷取的支援。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
  • Firefox 78 是最後一個可供 macOS 版本 10.9, 10.10, 10.11 使用的主要更新。若您使用這些版本，接下來幾年的更新將由 Firefox ESR （長期支援版本） 78.x 接續提供。
