Firefox 新鮮事

  • 全新功能

    對 Firefox 119 使用者逐步釋出包含更多內容的 Firefox View。你現在可以看到所有視窗的已開啟分頁，以及來自其他裝置的分頁（如有啟用「開啟分頁」同步）。現在也會列出瀏覽記錄，可以按日期或按網站排序。一如過往，「最近關閉的分頁」也列在 Firefox View 中。

    若要使用 Firefox View，請點擊分頁列最左方的「資料夾」圖示。

  • 全新功能

    對 Firefox 119 使用者逐步釋出：在編輯 PDF 時，除了加入文字和繪畫之外，現在還可以新增圖片和替代文字。

  • 全新功能

    「最近關閉的分頁」現在會在未啟用「開啟先前的視窗與分頁」時保留。手動「回復先前的瀏覽狀態」仍將重新打開任何先前開啟的分頁與視窗。

  • 全新功能

    如果你從 Chrome 匯入資料，Firefox 現在提供匯入部分擴充套件的功能。

  • 全新功能

    作為 全方位 Cookie 保護 的一部分，Firefox 現在支援 Blob URL 的分隔儲存，以減少第三方代理可用來追蹤個人的潛在要素。

  • 全新功能

    在「加強型追蹤保護」的嚴格模式下，網站可見的字體限制在系統字體和語言包字體等，以減緩字體指紋辨識。

  • 全新功能

    更新 Storage Access API 網路標準，以提高安全性，同時減少網站故障，並使 Firefox 向淘汰第三方 Cookie 再進一步。

  • 全新功能

    加密客戶端問候 (Encrypted Client Hello, ECH) 現在可供 Firefox 使用者使用，提供更隱私的瀏覽體驗。ECH 將 TLS 連線時的加密擴展到更多的握手步驟，並更好地保護敏感欄位。到 Mozilla Distilled 進一步閱讀 ECH 釋出的更多資訊。

  • 全新功能

    多媒體探測（Media sniffing）不再用於 application/octet-stream 類型的檔案，現在會下載這些檔案而非嘗試播放。

  • 全新功能

    在 Windows 上，如果在 滑鼠設定 中啟用「打字時隱藏指標」，則輸入時將會隱藏滑鼠指標。

  • 全新功能

    Firefox 現在提供桑塔利文（Santali, sat）版本。

  • 修正

    修復了導致在 Facebook 上捲動位置時意外跳動的問題。

  • 修正

    多項安全性修正。

  • 企業版

    你可以在 Firefox 119 企業版版本資訊中找到政策更新和企業版相關的錯誤修復資訊。

  • 開發功能

    Firefox 119 開發者資訊

  • 開發功能

    對非作用 CSS 樣式（Inactive CSS styles）功能進行數個改進。此功能協助辨識對元素沒有影響的 CSS 屬性。現在完整支援諸如 ::first-letter、::cue 和 ::placeholder 等偽元素。

  • 開發功能

    JSON 檢視器在 REST API 除錯時特別有用，它會以格式化的樣式顯示回應的 JSON 內容。現在，如果 JSON 無效或損毀，會自動切換回原始資料檢視，改善使用者體驗。

  • 網路平台

    現在支援 Custom Elements 的簡單屬性（simple attributes）和預設可及性語意（default Accessibility Semantics）的 ARIA reflection 。請注意，這包括布林值、枚舉、數字和字串屬性，但不包括參考其他元素的屬性。

  • 網路平台

    Cross-Origin-Embedder-Policy 現在支援 credentialless。

  • 網路平台

    CSS attr() 函數現在支援帶入一個預設參數，例如 attr(foobar, "Default value")。

  • 網路平台

    透過使用 Object.groupBy 或 Map.groupBy 方法，現在更容易對陣列（和可迭代物件）中的項目進行分組。

  • 請參考此版本的完整變更清單。 你可能也想瞭解前一個版本的更新。
