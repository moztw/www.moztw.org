Firefox 新鮮事

  • 參考 Firefox 新鮮事。
  • 修正 顯示推薦附加元件時，錯誤地發送請求至 addons.mozilla.org（bug 1526387）。
  • 修正 改善 Netflix 互動影片播放（bug 1524500）。
  • 修正 macOS 色彩管理不正確（bug 1506495）。
  • 修正 特定情形下「清除最近的歷史記錄」視窗大小不正常（bug 1523696）。
  • 修正 呼叫 WebRTC 時的影音延遲問題（bug 1521577 和 bug 1523817）。
  • 修正 某些 WebRTC 呼叫會造成影片不正確縮放（bug 1520200）。
  • 修正 在 proxy server 內使用 HTTP/2 WebSocket 發生的循環 CONNECT 請求狀況（bug 1523427）。
  • 修正 部分 Linux 發行版中 Enter 鍵在密碼欄位沒有效用（bug 1523635）。
  • 修正 各種穩定性與安全性修正。
  • 開發功能 適應性設計模式支援 <meta> viewport 標籤，最初於 Firefox 64 導入，現預設為關閉（bug 1521814），若要回復舊版行為，可於 about:config將 devtools.responsive.metaViewport.enabled 改為 true。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
