Firefox 新鮮事

  • 全新功能

    Firefox 現在將提供「暫時授予網站特定權限」的選項（例如地理位置）。臨時權限將在一小時後或關閉分頁時移除。

  • 全新功能

    現在滑鼠游標指向背景分頁時，Firefox 會顯示分頁預覽，讓你無需點入就能輕易的找到所需的頁面。

  • 全新功能

    在建議預設的翻譯語言時，Firefox 現在會考慮你之前使用過的語言。

  • 全新功能

    當搜尋欄為空時，我們重新引入透過 shift-enter 或 shift-click 來跳到搜尋引擎首頁的功能。

  • 修正

    多項 安全性修正。

  • 變更

    Firefox 翻譯 現在開始支援以下語言：

    • 瑞典文

  • 變更

    列出所有分頁選單 現在有了煥然一新的圖示。

  • 企業版

    你可以在 Firefox 131 企業版版本資訊 中找到政策更新和企業版相關的錯誤修復資訊。

  • 開發功能

    Firefox 131 開發者資訊

  • 開發功能

    作為 SVG 2 當前持續更新的一部分，SVGGraphicsElement.nearestViewportElement 和 SVGGraphicsElement.farthestViewportElement 已被正式移除。這些屬性自二月以來就已經顯示棄用警告。

  • 開發功能

    Firefox 現在支援「文字片段」（text fragments），讓使用者可以透過特殊的 URL 語法，直接連結到網頁中的特定文字。（深入瞭解）

  • 網路平台

    SameSite=None 的 Cookie 如果未包含 Secure 屬性，現在將被拒絕。

  • 網路平台

    現已支援「獨立分區狀態」（CHIPS）的 Cookie，允許開發者將 Cookie 儲存在按頂層網站個別區隔的儲存空間中。（深入瞭解）

  • 請參考此版本的完整變更清單。 你可能也想瞭解前一個版本的更新。
