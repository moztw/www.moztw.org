Firefox 3 Beta 3 新鮮事

發行日期： 2008 年 2 月 12 日

Firefox 3的Beta 3版本是Mozilla給開發者預覽測試的Firefox瀏覽器新版本，僅供測試使用。

Beta版本釋出的目的是讓網站開發者及測試社群能在下一個釋出階段前能得到使用者的回饋。當我們認為產品的品質演進到一定水準時，我們才會對使用者釋出最終的版本。然而，使用者如果安裝最新版本的Firefox，必須要有附加元件無法正常在beta版本使用的心理準備。

Beta的版本可以讓使用者對Firefox 3的模樣及功能有更清楚的認知，但是仍有新功能尚未包含在這個測試版本中，而且目前看到的許多部分仍會修改補強。

如有發現任何問題，我們非常歡迎您到 Mozilla Taiwan 討論區 與我們參與討論或是直接到 Bugzilla 回報錯誤訊息。


What's New in Firefox 3

Firefox 3 Beta 1 以新版 Gecko 1.9 網頁佈局引擎構成。研發此全新的引擎耗時 27 個月、更改了近兩百萬行程式碼，亦改善了超過一萬一千個大小問題。Gecko 1.9 為提升效能、穩定度、正確性，重構了部份程式架構，因此 Firefox 3 更加安全、更容易使用、自訂更輕鬆，同時提供許多給網 頁及擴充套件開發員的內部修改。

[Beta 3 新功能！] 根據前一個版本測試社群的回饋，Firefox 3 Beta 3 做了約1,300處的修改，包含提升穩定度及效能，調校記憶體使用，開發平臺改良及使用者介面的進化。

更安全
  • [Beta 3 新功能！] 網站安全資訊：點選位址列網站圖示即可查閱此網站的識別資訊，該網站的驗證資料會在點選後以輕鬆易懂的方式呈現。在之後 的版本裡，SSL 延伸驗證（Extended Validation）資訊也將一併顯示。 (Try it here!)
  • [Beta 3 新功能！] 阻擋有害軟體：當使用者到達某個已知會安裝病毒、間諜軟體、木馬等有害軟體時，有害軟體阻擋功能將警告使用者。您可在此測試（註：有害軟體網站黑名單 目前尚未正式啟用）
  • 阻擋偽造網頁：新版裡不會顯示偽造網頁的內容，您可在此測試。
  • 新的 SSL 錯誤資訊頁面：當 Firefox 碰上無效的 SSL 驗證資訊時，將出現更加簡潔的錯誤資訊頁。
  • 附加元件安全更新：為增強附加元件更新時的安全性，無法提供安全連線更新的套件將予以停用。
  • 整合防毒軟體： Firefox 會與呼叫您安裝的防毒軟體同對下載的檔案掃毒。
  • Vista 家長控制：Firefox 現在可依據 Windows Vista 系統上的家長控制設定來停用檔案下載功能。
  • 增加有效頂級域名服務 (Effective top-level domain, eTLD)，更能確保 Cookies 及其他受限內容僅在同一網域名稱下運作。
  • 更有效的防範 cross-site JSON data leaks.
更容易使用
  • 密碼儲存：登入網站時改以資訊列方式提示儲存密碼，這麼一來您可以確定登入成功再儲存。
  • 簡化附加元件安裝程序：不再強制使用附加套件的「白名單」安裝程序，如此在其他協力網站上安裝套件將可以少按幾次鍵。
  • [Beta 3 新功能！] 新的下載管理員: 更新後的介面可方便存取既有的下載檔案。
  • 支援檔案下載續傳功能：您現在可以在重新啟動 Firefox 或是重新設定連線後恢復檔案下載。
  • 全頁面縮放：改良後的頁面縮放功能，可以對頁面中的文字、圖片、功能表同時縮放，不同於 Firefox 2 只能對文字縮放
  • [Beta 3 新功能！] 播客（Podcasts）及 Videocasts 可以與錄音工具連結
  • 分頁工具列：當開啟較多的分頁時，可以利用滑鼠滾輪來捲動分頁工具列
  • 儲存瀏覽現況：當退出 Firefox 時可以選擇紀錄當下開啟的頁面
  • 改良「在分頁中開啟」功能：將書籤資料夾開啟至分頁時會全數開啟於新分頁，現有的分頁依然可以保留。
  • 網址列與搜尋列可以自訂調整寬度
  • 改善文字選取方式：搭配 Ctrl（Cmd）鍵可選取不連續文字區段、「雙點滑鼠並拖曳」能以「字」（而非字母）為單位快速選取相鄰文字、點三下滑鼠鍵更可選取整個段落。
  • 尋找工具列：尋找工具列開啟時，將自動含括目前選取區的文字。
  • Plugin 管理頁面：您可以在附加元件中關閉、開啟所安裝的 Plugin
  • 整合 Vista：選單採用 Vista 原生佈景主題。
  • [Beta 3 新功能！] 整合 Mac：使用 OS X 的拼字檢查工具，並支援以 Growl 提示已完成的下載及更新通知
  • [Beta 3 新功能！] 整合 Linux ：Firefox 可以使用 Linux 原生的 GTK 佈置主題
自訂更輕鬆
  • [Beta 3 新功能！] 星星按鈕：網址列加入星星圖示，按一次圖示可以快速的加入書籤；再按一次可以編輯相關書籤資訊
  • 標籤功能：為書籤加上關鍵字，方便你以其主題排序及搜尋
  • [Beta 3 新功能！] 在位址列裡輸入網頁的標題或標籤，即可自瀏覽歷史中查找要看的網頁。您可從建議清單中的網頁圖示、書籤標示、標籤標示知道此建議由何處而來
  • 智慧「Places」資料夾：有了書籤工具列中的智慧資料夾「Places」，您可快速查閱最常閱讀或最近加入書籤、標籤的網頁。組織書籤及瀏覽歷史：使用多種檢視模式及智慧資料夾，搭配更好的搜尋功能在書籤及瀏覽歷史中查找資訊
  • 整合網際應用程式：可以用網際應用程式（像是 Gmail）處理各式協定的連結，代替一般的單機軟體。網際應用程式需先行在 Firefox 中登記其協定方能使用
  • [Beta 3 新功能！] 下載及安裝附加元件: 可以透過附加元件管理員(Tools > Add-ons)從附加元件社群網站的數千種選擇中下載及安裝客制化的附加元件。當首次開啟附加元件管理員時，也會看到推薦的附加元件清單
  • 自訂下載後動作：新的「應用程式」設定面板，可供調整各式檔案類型及協定的處理方式
開發平台改良
  • 更新圖像及字型掌控方式：Gecko 1.9 擁有新的圖片及文字繪製架構，使得 CSS、SVG 及字型顯示方面都有長足進步。
  • 色彩管理：Firefox 現在能以圖片中嵌入的色彩資訊調整圖片色彩。（此功能需以 about:config 設定gfx.color_management.enabled 並重新啟動瀏覽器方能啟用。)
  • 離線應用程式：提供離線使用網際應用程式的能力。（當然，網站作者必須提供相對應的支援，一般使用者才能享用此功能。）
  • 各項內部功能概觀的完整清單可參考 Firefox 3 for developers。
改善效能
  • 更可靠：使用者的書籤、瀏覽歷史、Cookies 及偏好設定現在存放在具安全異動能力（transaction）的資料庫格式中，即使系統當機也能避免資料流失。
  • [Beta 3 新功能！] 更快速：大幅調整架構（使用 Cairo、重寫網頁佈局程序）提升程式效能，之後的測試版裡效能還會再提升。
  • [Beta 3 新功能！] 更有效使用記憶體：處理了 350 個以上的記憶體流失問題，再由新的 XPCOM 資源回收 功能預防此類問題。開發者正持續（以快速釋放快取物件的方式）改善記憶體使用方式、減少記憶體碎裂。

下載與安裝

系統需求

在安裝 Firefox 前請先確定您的電腦合乎 系統需求。

下載 Firefox 3 Beta 3

Mozilla.org 提供了 Windows、Linux 及 Mac OS X 平台上多種語言版的 Firefox 3 Beta 3，您可由此下載最新的 Firefox 3 Beta 3。若想自行編譯其他不支援的平台或語言，請見本文件最末節。

安裝 Firefox 3 Beta 3

請注意：安裝 Firefox 3 Beta 3 將覆蓋您現存的 Firefox 程式，這並不會造成書籤或瀏覽記錄遺失，但可能會有些附加元件要更新才能繼續使用。

移除 Firefox 3 Beta 3

在 Windows 作業系統中，您可以自「開始」選單中的「控制台」移除 Firefox 3 Beta 3。OS X 的使用者請移除「Firefox」 程式、Linux 使用者則請移除「firefox」 資料夾。

移除 Firefox 3 Beta 3 並不會將書籤、瀏覽記錄、擴充套件和其他附加元件一併刪除，這些資料會留在您的「個人設定資料夾」中，此資料夾存放位置則視作業系統而定：

Windows Vista Users\<使用者名稱>\AppData\Roaming\Mozilla\Firefox
Windows 2000, XP Documents and Settings\<使用者名稱>\Application Data\Mozilla\Firefox
Windows NT WINNT\Profiles\<使用者名稱>\Application Data\Mozilla\Firefox
Windows 98, ME Windows\Application Data\Mozilla\Firefox
Mac X ~/Library/Application Support/Firefox
Linux 及 Unix 系統 ~/.mozilla/firefox

移除 Firefox 3 Beta 3 後所安裝的任一版本 Firefox 將繼續使用個人設定資料夾中的資料。

擴充套件及佈景主題

Firefox 3 Beta 3 安裝的擴充套件可能已不相容於先前的版本而應加以升級，如發生此類問題，請回報附加元件原作者。一旦安裝 Firefox 3 Beta 3，所有擴充套件及佈景主題會先停用，待 Firefox 3 Beta 3 偵測出此套件無相容問題或能升級才會啟用。

已知問題

以下列出部分 Firefox 3 Beta 3 已知問題，請閱讀後再回報新的程式錯誤。

所有系統
  • Windows Live Mail 無法使用; 必須切換到 Hotmail 舊版介面 (bug 396259)
  • 新版的 GMail 聯絡人無法顯示 (bug 415252)
  • 新版 Yahoo! Mail 錯誤更新中，使用者暫時切換到舊版的介面 (bug 398381)
  • 無法在智慧書籤中拖曳書籤，現在只能利用剪下、貼上步驟來移動書籤 (bug 405198)
  • MathML 功能在 Firefox 3 Beta 3 尚未完全穩定 (bug 324857 and bug 363240)
  • 新的網址列，當使用 Window-Eyes 時加入書籤對話框與下載管理員通知的訊息不一致，使用 JAWS 8.0 或其他螢幕閱讀程式來測試 Firefox 3 Beta 3 (bug 393398)
  • 續傳檔案系統會因為伺服器不允許重新連線時發生錯誤 (bug 401846)
  • 在安裝部份的附加元件後，"重新啟動"的按鈕不會顯示 (bug 414936)
  • 部份附加元件在停用或移除後仍會被載入 (bug 406118)
  • 使用 Internet Download Manager 附加元件會使 Firefox 當掉，建議關閉或是移除該套件 (bug 382356)
  • 個人隱私 > 瀏覽紀錄 > 保留瀏覽紀錄裡的網址設定為 0 時沒有作用 (bug 366075)
Windows 及 Windows Vista
  • Windows Vista 及某些其他版本Windows並沒有提供一個Windows Media Player (WMP) 的plugin。必須透過一些步驟才能安裝plugin及開啟Windows Media檔案。安裝完畢後可能也需要做Windows Updates才能讓plugin正常地顯示這些檔案。
  • 當物件被拖放到書籤工具列時，資料夾不會自動展開。(bug 337761)
  • 同一時間利用下載管理員下載許多檔案後，AVG Virus 進行掃毒時會造成 Firefox 當掉 (bug 412565)
Mac OS X
  • 當使用 Google 文件時 Firefox 無法回應按鍵輸入 (bug 413882)
  • 上一頁、下一頁的選單尚不完善 (bug 415444)
  • Flash Player r60 以上本版存在一些問題 (bug 395983)
  • 建立新設定資料夾時, 可選擇性的匯入 Internet Explorer 資料。 從 Safari 匯入資料後， 書籤資料會錯置於 "由Internet Explorer" 資料夾 (bug 413021)
  • 在非 en-US 的語系下，應用程式資料夾的圖示會遺失 (bug 416055)
Linux 及 Unix
  • 在某些散佈版裡，逛某些網站時無法列印 (bug 414314)
  • 使用較舊散佈版的使用者需要更新認證才能送出當機報告 (bug 407748)
  • 瀏覽 SMB 分享使用較舊的 Gnome VFS 版本無法列出使用非 ASCII 字碼的檔案 (bug 255949)
  • AT-SPI 事件無法觸發當焦點再次回到 Firefox (bug 415919)

疑難排解

  • 設計不良或不相容的套件將導致瀏覽器發生問題，發生的問題包含當機或網頁瀏覽速度變慢等等。如果您碰上瀏覽器無法開啟、視窗顯示不正常等等狀況，便可能是擴充套件或佈景主題的問題。這時，請將瀏覽器重新啟動在安全模式：
    • 在 Windows 下用「Safe Mode」捷徑或在命令列中輸入 firefox.exe -safe-mode 指令。
    • 在 Linux 下請執行./firefox -safe-mode 指令。
    • 在 MacOS X 下請輸入

      cd /Applications/Firefox.app/Contents/MacOS/
      ./firefox-bin -safe-mode

    當啟動安全模式時，所有的附加元件將停用並使用預設的佈景主題。在安全模式下移除有問題的擴充套件/佈景主題，然後再正常的啟動瀏覽器。
  • 如果您想在移除某擴充套件後，以 -install-global-extension 指令將同一擴充套件裝到每個使用者設定檔中，則必須在重新安裝前先重開瀏覽器一次，以便徹底移除擴充套件。如果您沒這麼做，則擴充套件清單可能會有衝突項目而不能為所有設定檔安裝。
  • 如果您遇到的怪問題與書籤、下載、視窗位置、工具列、瀏覽記錄或其他設定項目有關，建議您在回報問題前先行建立新的使用者設定檔，並查看能否重現該問題。建立新設定檔的方式如下：執行 Firefox 時加上 -P 的命令列參數，選擇「管理使用者設定檔」按鈕，然後選擇「建立使用者設定檔」。將各種設定(書籤、儲存的密碼等)從舊的設定檔中分次轉移到新的設定檔，每移轉一項便檢查是否發生問題。如果您發現某特定設定檔資料會造成問題，請回報錯誤並且附上該檔案。

常見問題集

  1. 我該怎麼幫忙呢？

    我們需要開發員和測試社群回饋建議，幫我們將 Firefox 變得更盡善盡美。請先閱讀錯誤回報注意事項，再到 Bugzilla 回報。您也可以使用回饋表單表達意見。

  2. 你們為什麼都沒有回信？

    請使用討論區， Firefox 團隊會定期閱讀上面的文章。我們常常都收到大量的信件，而您的可能不小心漏失了。

  3. 我該去哪裡取得新的佈景主題與擴充功能 (附加元件)？

    擴充套件及佈景主題可自 Mozilla 附加元件中心下載。

  4. Firefox 3 Beta 3 是哪些人做出來的？

    很多人。「說明＞關於 Mozilla Firefox」中有份名單，列出部分曾對 Firefox 3 Beta 3 有所貢獻的人。

  5. Firefox 3 Beta 3 的原始碼在何處？

    Firefox 3 Beta 3 原始碼已開放下載，而目前開發中的最新程式碼則需以 CVS 取得。Firefox 相關程式為「mozilla/browser」、「mozilla/toolkit」及「mozilla/chrome」，請參考建製步驟。

  6. 電子郵件程式呢？

    Firefox 3 Beta 3 能與您預設的任何郵件程式協同運作， 而我們推薦您試試我們下一代的郵件軟體：與 Firefox 合作無間的 Mozilla Thunderbird。

非官方建製版

Mozilla 代表 各國作者提供了各國語系的地方化版本，您可至Firefox 3 Beta 3 下載頁取得檔案。

其他尚未被官方 Firefox 3 Beta 3 驗證通過的本地化版本可以在 FTP 站台取得。
