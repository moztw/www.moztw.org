Firefox 新鮮事

  • 全新功能 根據瀏覽習慣推薦 Firefox 功能、服務、附加元件（僅 US 版本有效）。
  • 全新功能 選取複數分頁，方便一次關閉、移動、加入書籤、釘選等。
  • 全新功能 在新增的 about:performance「工作管理員」中檢視各分頁使用能源的狀況，也可以直接關閉分頁。
  • 全新功能 利用 Clang LTO (Link Time Optimization) 改善 Mac 和 Linux 上的效能。
  • 全新功能 Windows: 直接在「頁面操作」(Page Action) 中分享目前網頁至其他裝置。
  • 全新功能 在附加元件的工具列按鈕上，可以直接移除元件。
  • 全新功能 企業版：更新 macOS 版 policy 引擎，允許以設定檔調整部署政策。
  • 修正 各種安全性修正。
  • 變更 移除 RSS 預覽和即時書籤功能，需要的使用者可安裝附加元件。
  • 變更 不再信任 Symantec 發行的 TLS 憑證。網站管理者應儘速撤換現存的 Symantec TLS 憑證。
  • 變更 改進 about:crashes 設計，清楚呈現錯誤報告傳送至 Mozilla 的時間，並說明本地端移除的錯誤仍會於 crash-stats.mozilla.com 保留。
  • 變更 macOS：用來增加 www 和 .com 至網址的鍵盤快速鍵，由 ⌘[cmd]-enter 改為 ctrl-enter。
  • 開發功能 CSS 格線檢測器支援同時啟用三層覆蓋格線顯示。
  • 開發功能 網頁主控台支援 JavaScript 語法上色。
  • 開發功能 滑鼠移至「輔助功能」面板中的元素時，會指出文字與背景色的對比率。
  • 開發功能 支援 CSS scrollbar 標準。
  • 開發功能 macOS 的 WebVR 支援。
  • 開發功能 參考開發者功能改進。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
