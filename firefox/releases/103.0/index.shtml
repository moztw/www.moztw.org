Firefox 新鮮事

  • 全新功能

    更換至更為現代的上鎖 API（modern lock API），提高了在 macOS 在高 CPU 負載期間的反應能力。

  • 全新功能

    你總是會忘記什麼嗎? PDF 表單中現在會突顯必填欄位。

  • 全新功能

    改進了高更新率顯示器(120Hz+)的性能。

  • 全新功能

    喜歡子母畫面的字幕功能嗎? 它變得更好了：你現在可以直接從子母畫面的視窗更改字幕字體的大小。此外它現在也可以在 Funimation, Dailymotion, Tubi, Hotstar 和 SonyLIV 上使用。

  • 全新功能

    現在可以使用 Tab、Shift+Tab 和 Arrow 鍵來存取工作列中的各項按鈕。更多詳細資訊可以查看此文章。

  • 全新功能

    Windows 的大文字輔助功能設定，現在會影響到所有 UI 及內容頁面，而不是應用於系統字體大小。

  • 全新功能

    開心! 現在在全新安裝的 Firefox 會固定到 Windows 10 和 11 的任務列上，往後可以更方便的使用 Firefox。（這也將允許 Firefox 在安裝後能更快的啟動。）

  • 修正

    修補多個安全弱點。

  • 變更

    刪除了設定選項中允許 SHA-1 簽名選項：不再支援證書中的 SHA-1 簽名（早已確定不再足夠安全）

  • 開發功能 Firefox 103 開發者資訊
  • 網路平台

    Firefox 新增 large, small, dynamic viewport units 及 (*vi 與 *vb) 等邏輯變數支援，提供使用者選擇讓網頁元素縮放到「最小」（動態工具列可視時）、「最大」（動態工具列隱藏時）或「變動」的 viewport 尺寸（根據動態工具列的現有狀態而定）。

  • 網路平台

    你的資訊現在透過預設啟用的「全面 Cookie 保護」增強了對於線上追蹤的保護。所有第三方 Cookie 現在都隔離到分離的儲存區當中。

  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
