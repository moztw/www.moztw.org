Firefox 新鮮事

  • 在 Mozilla，我們相信您有隱私權。您不該在上線時被追蹤，不論您正在查看銀行餘額、尋找最好的醫師或者買鞋子，不道德的追蹤公司不該有辦法在您瀏覽網路時追蹤您。因此，我們持續強化 Firefox，以替我們的使用者抵抗線上追蹤。
  • 全新功能 Firefox 現在 支援 以子母畫面模式 同時觀賞數個影片。
  • 全新功能 今天，Firefox 在加強型追蹤保護的「嚴格」模式中加入了 Total Cookie Protection 功能。在 Total Cookie Protection 裡，每個網站都會有它自己的「餅乾罐（cookie jar）」，在您瀏覽不同網站時，避免 Cookie 被用於跨網站追蹤。
  • 全新功能 我們現在改善了網頁列印功能，呈現更乾淨的設計，並與您電腦上的印表機設定整合的更好了。

  • 全新功能 現在加拿大的 Firefox 使用者可以使用 信用卡管理及自動填入功能 了
  • 修正 閱讀模式現在可以開啟本機上的 HTML 頁面。
  • 修正 使用螢幕閱讀器 quick navigation 來快速移動到可編輯的文字控制區塊時，再也不會於某些 grid 中錯誤的移動到無法編輯的文字區塊，比方說在 messenger.com 上時。
  • 修正 Orca 螢幕閱讀器的游標預覽（mouse review）功能，現在在 Firefox 切換分頁後可以正常運作了。
  • 修正 在有橫跨多個欄位的儲存格的表格上，螢幕閱讀器回報欄位標題時再也不會發生錯誤。
  • 修正 閱讀模式上的連結現在有更佳的顏色對比度。
  • 修正 修補幾種不同的安全弱點。
  • 變更 在 Linux 和 Android 上，阻擋 stack clash attack 的防護功能已被啟動。
  • 變更 從 Firefox 86 開始，WebRTC PeerConnections 已經不再支援 DTLS 1.0 了。從現在起，所有 WebRTC 服務將必須要至少支援 DTLS 1.2。
  • 變更 所有影像解碼都被整合到 RDD 程序中，這使得 Firefox 更加安全。
  • 企業版 多個 Bug 修復與新政策加入到最新的 Firefox 版本。您可在 Firefox 86 企業版版本資訊找到更多資訊。
  • 開發功能 Firefox 86 for developers 開發者資訊
  • 開發功能 CSS: CSS 中的 image-set() 函數已經啟用，讓 CSS 中圖片的反應可以更快。
  • 開發功能 非作用中的 CSS 工具上，現在在你設定 internal table element 的 margin 或 padding 屬性時會顯示警告訊息。

  • 開發功能 開發者工具箱現在會顯示當前的頁面上總共有幾個錯誤。這可以加速訊息浮現，讓開發者得知當前的頁面可能有些問題。點擊紅色驚嘆號，會將使用者帶到 Console 面板。

  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
