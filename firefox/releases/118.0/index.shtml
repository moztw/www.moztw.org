Firefox 新鮮事

  • 全新功能

    現已為 Firefox 使用者提供網頁內容的自動翻譯功能！與雲端為基礎的替代產品不同，翻譯是在 Firefox 中本地進行的，因此翻譯的文本不會離開您的電腦。感謝歐盟 Bergamot 研發專案的各個合作夥伴。

  • 全新功能

    Firefox 中的 Web Audio 現在在所有系統上使用 FDLIBM 數學函式庫，以提高指紋保護的匿名性。

  • 全新功能

    在為了緩解隱私視窗中的字體指紋識別，網站對字體的可見性已經限制在系統字體和語言包字體。

  • 全新功能

    現在 Firefox 使用者可以在 Google Meet 上使用視覺效果和背景模糊！（附註：這些效果也已經向前相容到 Firefox 115。）

  • 全新功能

    Firefox Suggest 使用者（目前僅限美國）現在可以直接在網址列中看到根據關鍵字提供的建議瀏覽器附加元件。

  • 修正

    多項安全性修正。

  • 企業版

    你可以在 Firefox 118 企業版版本資訊中找到政策更新和企業版相關的錯誤修復資訊。

  • 開發功能

    Firefox 118 開發者資訊

  • 網路平台

    現在支持 10 個新的 CSS 數學函數：round、mod、rem、pow、sqrt、hypot、log、exp、abs、sign。

  • 網路平台

    現在已經預設啟用 OpaqueResponseBlocking。

  • 網路平台

    開始支援 <search> 元素。 <search> 元素是一個群組元素，用以容納所有用於搜尋或篩選操作的元素。

  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
