Firefox 新鮮事

  • 參考 Firefox 新鮮事。
  • 修正 安全性修正。
  • 修正 選用特定 Windows 安全政策（例如 Windows Defender 惡意探索防護或 Webroot）時，Firefox 58.0 載入網頁的問題（bug 1433065）。

已知問題

  • 尚未修復 使用 Windows 遠端桌面協定（RDP）執行的 Firefox 可能因為縮限的安全限制而無法播放聲音。在下個修正之前，可參考此問題的暫時性解法。
  • 尚未修復 使用螢幕閱讀器時可能有效能問題，建議在新版修正前，先暫時使用 Firefox ESR 版本。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
