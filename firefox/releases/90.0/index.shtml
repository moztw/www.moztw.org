Firefox 新鮮事

  • 全新功能

    向全新的 Firefox 説嗨！為了讓你更快抵達想去的地方，我們重新設計了核心體驗，現代化、更加迷人、容易使用。

    從 Firefox 89 開始，你會注意到以下變化：

    簡化的瀏覽器介面與工具列： 移除較少用的項目，聚焦在瀏覽時更重要的元件。

    乾淨、簡化的選單：根據使用狀態重新排序選單內容。更新標籤文字，並移除圖示。

    更新提示框（prompts）：資訊列（infobar）、面板（panel）與對話框（modal）獲得更乾淨的設計與明確的用語。

    啟發的分頁設計：浮出的分頁內置整齊的資訊，與在你需要時出現的介面提示，例如聲音控制器的視覺指引。四方形的設計凸顯焦點，提示你可以隨所需輕易的移動分頁。

    更少干擾：減少警示與訊息的數量，讓你瀏覽時更加專心。

    更凝聚、安靜的視覺：較淡的圖示、改善的色彩組合、更加一致的樣式。

    本版本同時強化了以下隱私功能：

    • 我們強化了 Firefox 瀏覽器的隱私瀏覽模式，加入全面 Cookie 保護（Total Cookie Protection）功能，將 cookie 限制在產生他們的網站上，避免企業使用 cookie 跨站追蹤你的瀏覽。這個功能原先於 Firefox 加強型追蹤保護的嚴格模式（ETP Strict mode）中提供。
  • 全新功能

    在 macOS 上，我們加入許多程式都有的彈性「捲過頭」效果（elastic overscroll effect），以溫和的彈跳動畫，來提示你已捲到頁面底部。

    此外，我們還加入了智慧縮放：雙指雙點觸控版，或在 Magic Mouse 上單指雙點，可縮放畫面並聚焦到游標下的內容。

  • 全新功能

    原生的右鍵選單：macOS 上的右鍵選單，現在改用原生的介面，並支援深色模式。

  • 全新功能

    在有 NVIDIA 二進位驅動程式的 Linux 桌面環境上啟用 WebRender。

  • 修正

    在廣色域的顯示器上，macOS 的 Firefox 顏色不再過度飽和。沒有標記的圖片會正確的以 sRGB 格式處理。標記為 sRGB 的圖片顏色能對應到 CSS 顏色。

  • 修正

    macOS 的全螢幕模式中，將滑鼠移到螢幕最上方時，分頁列不再會被系統選單蓋住。

  • 修正

    也是在 macOS 的全螢幕模式中，現在可以隱藏工具列，以獲得更具沈浸感的全螢幕體驗。這讓 macOS 的體驗與 Windows 及 Linux 版看齊。

  • 修正 修補幾種不同的穩定性與安全弱點。
  • 變更

    引入非原生的網頁表單控制項實作，提供更現代的設計，部分改善網頁載入速度。監控會影響表單控制項大小或樣式猜測的網頁版面問題。

  • 變更

    畫面截圖功能現在已移到右鍵選單內。你也可以在工具列新增「畫面截圖」按鈕。深入瞭解。

  • 企業版 在最新的 Firefox 修復多個 Bug 與加入新政策。您可在 Firefox 89 企業版版本資訊找到更多資訊。
  • 開發功能 Firefox 89 開發者資訊
  • 開發功能

    在檢測器面板中可編輯的 BoxModel 屬性上提供更好的鍵盤導覽功能。

  • 網路平台

    開始支援 Event Timing API。

  • 網路平台

    開始支援 CSS forced-colors media query。
