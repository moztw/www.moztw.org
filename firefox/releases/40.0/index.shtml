Firefox 新鮮事

  • 全新功能 支援 Windows 10。
  • 全新功能 增加對不安全軟體（垃圾軟體）下載的防護。
  • 全新功能 根據最近瀏覽的網站，在「新分頁」顯示推薦磚塊（限 en-US 語系）。
  • 全新功能 Hello 可以將連結加入通話，作為關於聊天的資訊。
  • 全新功能 更新「附加元件管理員」外觀，採用和新版選項一致的設計。
  • 全新功能 藉由 OMTC (Off Main Thread Compositing) 改善捲動、繪圖和影片播放的效能（限 GNU/Linux）。
  • 全新功能 改善顯卡黑名單機制：藉由指定 Firefox 版本範圍縮減裝置被擋下的情形。
  • 變更 未經 Mozilla 驗證的附加元件會顯示警告訊息。
  • 變更 藉由 asynchronous initialization 改善 NPAPI 外掛程式的效能。
  • 變更 藉由硬體 vsync 達成更順暢的動畫和捲動效能（限 Windows）。
  • 變更 更快速的 JPEG 圖片繪製，縮放時使用更少記憶體。
  • 變更 子文件中的資源不再進行 HTTP 認證請求，避免使用者無意間透露登入資料。
  • HTML5 IndexedDB 預設行為改為非 durable transaction。
  • HTML5 實作 AudioBufferSourceNode.detune，模組化以 cent（衡量音樂時距的對數單位）為單位的播放。
  • 開發功能 改善「效能」工具：Waterfall 檢視、Call Tree 檢視和 Flame Chart 檢視。
  • 開發功能 檢測器「規則」檢視新增方便調節 CSS filter 的浮動面板。
  • 開發功能 網頁主控台會顯示來自 SharedWorker 和 ServiceWroker 的 Console API 訊息。
  • 開發功能 新增「顯示頁面上的尺規」功能，提供輕便的水平、垂直測量工具。
  • 開發功能 檢測器搜尋會包括頁面上所有頁框的內容。
  • 修正 修正卡納達語（Kannada）文字在內建 PDF 檢視器的顯示問題。
  • 修正 各種安全性修正。
  • 請參考此版本的完整變更清單。 您可能也想瞭解 前一個版本的更新。

已知問題

  • 尚未修復 在附加元件安裝通知出現時重啟 Firefox，私密瀏覽進行中的下載可能被無預警地取消（bug 1185294）。
