Firefox 新鮮事

  • 全新功能

    現在信用卡自動填充功能已擴展到 IT、ES、AT、BE 和 PL 語系的 Firefox 使用者。

  • 全新功能

    macOS 用戶現在可以在 about:preferences 內切換是否使用 Tab 在表單控制元件和鏈接間移動焦點（tabability）。

  • 全新功能

    在那些擁有自訂行為的網站上，為了避免在按下 shift+右鍵 時出現非預期結果，Firefox 現在有一個 dom.event.contextmenu.shift_suppresses_event 偏好設置，可以停止右鍵選單出現。

  • 修正

    使用螢幕閱讀器來瀏覽時，YouTube 影片列表時現在可以正確捲動。

  • 修正

    多項安全性修正。

  • 變更

    在 Wayland 桌面環境中，Firefox 不再顯示自己的螢幕分享指示。將使用系統預設的分享指示。

  • 企業版

    你可以在 Firefox 117 企業版版本資訊中找到政策更新和企業版相關的錯誤修復資訊。

  • 開發功能

    Firefox 117 開發者資訊

  • 開發功能

    開發者工具的相容性面板，現在增加了新的 CSS 相容性資訊。有相容性問題的屬性旁邊會出現圖示。當游標指向圖示時，工具提示會指出哪些瀏覽器不支援。且該屬性會連結到對應的 MDN 頁面，以便使用者了解更多資訊。

  • 開發功能

    如果啟用了「保留紀錄」選項，console.clear() 現在不再會清除主控台輸出。

  • 網路平台

    現在已預設啟動改進 CSS 嵌套（improved CSS nesting）支援。

  • 網路平台

    Firefox 現在支援 RTCRtpScriptTransform。

  • 網路平台

    現在支援 ReadableStream.from，允許從（非同步的）可迭代物件建立 ReadableStream。

  • 網路平台

    Firefox 現在支援 math-style 和 math-depth CSS 屬性，以及 font-size: math 值。

  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
