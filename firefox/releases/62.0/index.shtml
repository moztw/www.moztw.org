Firefox 新鮮事

  • 全新功能 Firefox 首頁可選擇呈現最多 4 列的熱門網站、Pocket 頁面或精選網站。
  • 全新功能 啟用「容器」的使用者，分頁選單會出現「使用容器開啟」以便進行切換。
  • 全新功能 在移除所有對 Symantec 發行憑證的信任前，在 about:config 提供 security.pki.distrust_ca_policy 選項讓使用者自行收回信任，設定為 2 便能撤除信任。
  • 全新功能 支援 FreeBSD WebAuthn。
  • 全新功能 Windows: 藉由 Parallel-Off-Main-Thread Painting 改善圖像描繪。
  • 全新功能 支援 CSS Shape，帶來更豐富的版面配置能力。同時 CSS 檢測器也提供了新的 Shape Path 編輯器。
  • 全新功能 支援 CSS Variable Fonts (OpenType Font Variations)，單一字型檔案就能提供美麗的文字。
  • 全新功能 Firefox 企業版更新：
    • AutoConfig 預設受到 API 文件所述的沙盒限制。 目前可將 general.config.sandbox_enabled 設為 false 來關閉沙盒，但長遠而言我們會移除關閉沙盒的功能。 若仍要以 AutoConfig 指令碼進行微調，請使用 Firefox Extended Support Release (ESR) 版本。
  • 全新功能 新增加拿大英語 (en-CA) 語系支援。
  • 變更 移除書籤的「描述」欄位。曾經手動編輯的使用者可將資料以 html 或 json 格式匯出，未來版本將刪除這些資料。
  • 變更 暗色佈景主題在 macOS 10.14 dark mode 中會自動套用。
  • 變更 調整 security.pki.name_matching_mode 選項的預設值為 3（強制允許）。
  • 變更 macOS: 在更安全的沙盒中執行 Adobe Flash，詳見說明。
  • 變更 中斷 Firefox Sync 後，提供選項以清除裝置上的個人資料（包括書籤、密碼、歷史、cookie 和網站資料）。
  • 變更 WebRTC 螢幕分享：共享視窗時，會將視窗移到前景。
  • 開發功能 三欄檢測器，可將「規則」分離為獨立面板。
  • 開發功能 參考開發者功能改進。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
