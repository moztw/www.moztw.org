Mac Firefox 4 正體中文版字體更新

你是不是發現，Mac OS X 平台上的 Firefox 4 看起來不太一樣了呢？

Mac Firefox 正體中文版的預設字體，隨著 Firefox 4 釋出同時已更換為「黑體-繁」。以下解釋為什麼這麼做，以及如何變更相關設定。


這裡要放上主題圖片

Mac 正體中文版的 Firefox 1.0 至 3.6.x 版，預設的正體中文字體是「蘋果儷中黑」，它自 2004 年 Firefox 釋出開始至今，已陪伴了我們將近七年的時間。

2009 年 8 月 Mac OS X 10.6 上市，新的系統預設中文字體「黑體-繁」隨之現身。但這個新字體極具爭議，包含筆畫、標點符號等都備受詬病。因此隨後發表的 Firefox 3.6 並未更換預設正體中文字體。

「黑體-繁」在釋出初期的許多令人詬病之處，在 OS X 10.6.3 中大多已完成修正，而相對於蘋果儷中黑及黑體 Pro 兩個 OS X 舊的系統中文字體，黑體-繁的優點相對不少，因此我們在 Firefox 4 RC 1 釋出前夕，開始考慮更換預設正體中文字體，並且向使用者提出徵詢。

調查過程與結果

調查的方式如下：我們將黑體-繁與黑體 Pro 作為預設字體的顯示效果，在四個網頁上抓圖（Yahoo 奇摩、MozTW Firefox 版本資訊、Mozilla Links 正體中文版、Wikipedia「黑體-繁」詞目 ），作成以下之比較投影片，並放置在此，於投影片的最後附上匿名線上問卷網址。

調查的連結（http://go.sto.tw/heiti4fx）在 3/13 凌晨於 MozTW 討論區、Mobile 01、PalmisLife、Ptt Mac 版及 Foxmosa 的 Plurk公開

截至 3/20 23:00 為止，共有超過 640 人點擊投影片網址，142 人完成投票。調查結果為：維持現狀（使用蘋果儷中黑）的票數為 45 票；更換為黑體-繁的票數為 97 票。比例約為 1:2，據此結果，我們向 Mozilla 提出更換 Firefox 預設正體中文字體的意見。（Bug 599289）

黑體-繁的優點

原先的預設字體「蘋果儷中黑」的主要問題在於只有一種粗細，且相較於其他字體粗上不少，在瀏覽網頁時，如果遇上需要使用粗體的場合，會造成文章內文黑成一團。

如何變更預設字體

如果不滿意要怎麼改回來，偏好設定及 about:config 都能提到。 行至「偏好設定→內容→預設字體」調整設定試試
